## 五、CCMR1 — 捕获/比较模式寄存器 1（核心）

### 5.1 寄存器结构

```
CCMR1（16位）分两半，各控制一个通道：

高 8 位（bit[15:8]）→ 通道 2 配置
低 8 位（bit[7:0]） → 通道 1 配置

每个 8 位的定义根据 CCS 选择不同而不同：
  CCS = 00 → 输出比较模式（OC 定义）
  CCS ≠ 00 → 输入捕获模式（IC 定义）
```

### 5.2 通道 2 输出模式关键位

| 位 | 名称 | 功能 | 本实验 |
|:--:|------|------|:------:|
| 9:8 | **CC2S** | 通道 2 方向选择 | **00** = 输出 |
| 12:10 | **OC2M** | 输出比较模式 | **110** = PWM 模式 1 |
| 13 | OC2PE | 输出比较预装载使能 | 0 或 1 |
| 15 | OC2CE | 输出比较清零使能 | 0（不用） |

### 5.3 CCS 位说明

```
CC2S = 00 → 输出比较模式
CC2S = 01 → 输入模式（IC 映射到 TI2）
CC2S = 10 → 输入模式（IC 映射到 TI1）
CC2S = 11 → 输入模式（IC 映射到 TRC）

⚠️ 注意：与 GPIO 的 CRL/CRH 相反！
  GPIO: MOD = 00 → 输入
  定时器: CCS = 00 → 输出
```

### 5.4 OCM 位（输出比较模式）

| OC2M[2:0] | 模式 | 本实验 |
|:---------:|------|:------:|
| 000 | 冻结 | — |
| 001 | 匹配时强制输出高 | — |
| 010 | 匹配时强制输出低 | — |
| 011 | 匹配时翻转 | — |
| 100 | 强制输出低 | — |
| 101 | 强制输出高 | — |
| **110** | **PWM 模式 1** | ✅ |
| 111 | PWM 模式 2 | — |

### 5.5 OC2PE — 预装载使能

```
OC2PE = 0：CCR2 写入后立即生效（写预加载 → 立即刷到影子）
OC2PE = 1：CCR2 写入后等更新事件才生效

CCR 也有影子寄存器（框图中有阴影）
一般不开启，写入后立即生效即可
```

## 六、CCER — 捕获/比较使能寄存器

### 6.1 通道 2 关键位

| 位 | 名称 | 功能 | 本实验 |
|:--:|------|------|:------:|
| 5 | **CC2P** | 通道 2 极性 | 0 = 高电平有效 |
| 4 | **CC2E** | 通道 2 输出使能 | **1** = 使能输出 |

### 6.2 CC2P — 极性说明

```
CC2P = 0：高电平有效（默认）
  → PWM 模式 1：CNT < CCR 时输出高电平
  → 高电平时间 = 占空比

CC2P = 1：低电平有效
  → PWM 模式 1：CNT < CCR 时输出低电平
  → 定义反转

本实验：
  LED 低电平点亮
  理论上应选低电平有效（CC2P = 1）
  但实际上用默认高电平有效（CC2P = 0）也能实现呼吸灯
  → 只要占空比变化，不管极性如何都能控制亮度
```

### 6.3 CC2E — 输出使能

```
CC2E = 0：通道 2 输出禁用
CC2E = 1：通道 2 输出使能（必须开启）

必须置 1，否则 PWM 信号无法输出到引脚
```

---

## 七、CCR2 — 捕获/比较寄存器 2

```
16 位寄存器，存放比较基准值

PWM 模式 1 中：
  → CCR2 决定占空比
  → 占空比 = CCR2 / (ARR + 1) × 100%

初始值可以任意设置（后续会动态修改实现呼吸灯）
```

---

## 八、完整配置流程

```
① 开启时钟
   ├── RCC->APB1ENR |= TIM5EN    → 定时器时钟
   └── RCC->APB2ENR |= IOPAEN    → GPIO 时钟

② 配置 GPIO（PA1 复用推挽输出）

③ 配置 PSC（预分频器）

④ 配置 ARR（自动重装载值）

⑤ 配置 CCR2（初始占空比）

⑥ 配置 CCMR1
   ├── CC2S = 00（输出模式）
   └── OC2M = 110（PWM 模式 1）

⑦ 配置 CCER
   ├── CC2P = 0（高电平有效）
   └── CC2E = 1（使能输出）

⑧ 使能定时器（CR1.CEN = 1）
```

## 九、PWM 参数计算

### 9.1 设计目标

```
PWM 频率 = 1kHz（人眼看不到闪烁）
分辨率 = 1%（100 级亮度调节）
```

### 9.2 计算过程

```
① PWM 周期 = 1 / 1kHz = 1ms
② 分辨率 1% → ARR + 1 = 100 → ARR = 99
③ 计数频率 = 100 × 1kHz = 100kHz
④ PSC = 72MHz / 100kHz - 1 = 719

验证：
  PWM 频率 = 72MHz / (720 × 100) = 1kHz ✅
  CCR = 0   → 占空比 0%   → LED 熄灭
  CCR = 50  → 占空比 50%  → LED 半亮
  CCR = 100 → 占空比 100% → LED 最亮
```

## 十、呼吸灯实现思路

```
CCR2 从 0 逐渐增加到 ARR → LED 渐亮
CCR2 从 ARR 逐渐减少到 0 → LED 渐暗
循环往复 → 呼吸效果

主循环伪代码：
while (1)
{
    for (CCR2 = 0; CCR2 <= ARR; CCR2++)   // 渐亮
    {
        TIM5->CCR2 = CCR2;
        Delay_ms(N);
    }
    for (CCR2 = ARR; CCR2 >= 0; CCR2--)   // 渐暗
    {
        TIM5->CCR2 = CCR2;
        Delay_ms(N);
    }
}
```

## 十一、GPIO 配置（PA1）

```
PA1 配置为复用推挽输出：
  CRL[7:4]（PA1 对应 CRL 的高 4 位）：
    CNF = 10（复用推挽输出）
    MODE = 11（50MHz 输出速度）
```

| 配置项 | 值 | 说明 |
|--------|:--:|------|
| CNF | 10 | 复用推挽输出 |
| MODE | 11 | 50MHz |
| 上下拉 | 无 | 推挽输出不需要 |

## 十二、CCR 预装载说明

```
CCR 也有影子寄存器（框图中有阴影）

┌──────────────┐
│ 预加载寄存器  │ ← CPU 写入 CCR2
└──────┬───────┘
       │ 更新事件（或立即，取决于 OC2PE）
       ↓
┌──────────────┐
│ 影子寄存器    │ ← 真正参与比较的值
└──────────────┘

OC2PE = 0：写入后立即生效
OC2PE = 1：等到更新事件后才生效

本实验：OC2PE = 0 即可（写入后立即改变占空比）
```

## 十三、寄存器操作汇总

| 步骤 | 寄存器 | 操作 | 含义 |
|:----:|--------|------|------|
| ① | RCC->APB1ENR | OR TIM5EN | 开启 TIM5 时钟 |
| ② | RCC->APB2ENR | OR IOPAEN | 开启 GPIOA 时钟 |
| ③ | GPIOA->CRL | 配置 PA1 | 复用推挽输出 |
| ④ | TIM5->PSC | = 719 | 720 分频 |
| ⑤ | TIM5->ARR | = 99 | 100 级计数 |
| ⑥ | TIM5->CCR2 | = 50 | 初始占空比 50% |
| ⑦ | TIM5->CCMR1 | OC2M=110, CC2S=00 | PWM 模式 1 |
| ⑧ | TIM5->CCER | CC2E=1, CC2P=0 | 使能输出，高电平有效 |
| ⑨ | TIM5->CR1 | CEN=1 | 启动定时器 |
