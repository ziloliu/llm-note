# STM32 通用定时器（TIM2~TIM5）— 时钟源详解 笔记

---

## 一、通用定时器概述

### 1.1 基本信息

| 特性 | 说明 |
|------|------|
| 编号 | TIM2, TIM3, TIM4, TIM5 |
| 数量 | 4 个 |
| 位宽 | 16 位 |
| 连接总线 | APB1 |
| 定时器时钟 | 72 MHz（APB1 36MHz × 2） |
| 功能定位 | **标准版**（覆盖基本定时器全部功能 + 扩展功能） |

### 1.2 与基本定时器功能对比

| 功能 | 基本定时器 (TIM6/7) | 通用定时器 (TIM2~5) |
|------|:-------------------:|:-------------------:|
| 定时中断 | ✅ | ✅ |
| 向上计数 | ✅ | ✅ |
| 向下计数 | ❌ | ✅ |
| 向上/向下交替计数 | ❌ | ✅ |
| 内部时钟源 | ✅ | ✅ |
| 外部时钟源（ETR） | ❌ | ✅ |
| 外部时钟源（通道输入） | ❌ | ✅ |
| 内部触发（定时器级联） | ❌ | ✅ |
| 输入捕获 | ❌ | ✅（4 个通道） |
| 输出比较 | ❌ | ✅（4 个通道） |
| PWM 输出 | ❌ | ✅ |
| 编码器接口 | ❌ | ✅ |
| 霍尔传感器接口 | ❌ | ✅ |
| 触发 DAC | ✅ | ✅ |
| 通道数 | **0** | **4** |

## 三、时钟源详解（4 种）

### 3.1 时钟源总览

| 时钟源 | 缩写 | 来源 | 引脚 | 使用频率 |
|--------|------|------|:----:|:--------:|
| **内部时钟** | CK_INT | APB1 × 2 = 72MHz | 无 | **最常用** |
| **外部时钟源模式 1** | TI1FP1 / TI2FP2 | 通道 1/2 输入 | CH1/CH2 | 较少 |
| **外部时钟源模式 2** | ETRF | 外部触发引脚 | ETR | 较少 |
| **内部触发** | ITR0~3 | 其他定时器输出 | 无（片内） | 级联时 |

### 3.2 内部时钟（CK_INT）— 默认

```
来源：APB1 总线时钟 × 2 = 36MHz × 2 = 72MHz

特点：
  → 最常用，大部分情况直接使用
  → 不需要任何外部连接
  → 默认时钟源，无需额外配置

使用场景：
  → 普通定时中断
  → PWM 输出
  → 输出比较生成方波
  → 绝大多数应用
```

### 3.3 外部时钟源模式 1

```
信号路径：
  外部引脚(CH1/CH2) → 输入滤波 → 边沿检测 → 触发控制器 → CNT

可用信号：
  → TI1FP1：通道 1 输入信号（滤波后）
  → TI2FP2：通道 2 输入信号（滤波后）

特点：
  → 只有通道 1 和通道 2 可以作为时钟源
  → 通道 3/4 不行
  → 信号经过滤波和边沿检测处理

使用场景：
  → 定时器级联（通过外部引脚连接）
  → 对外部脉冲信号计数
  → 测量外部信号频率
```

### 3.4 外部时钟源模式 2

```
信号路径：
  ETR 引脚 → 极性选择 → 边沿检测 → 预分频 → 输入滤波 → 触发控制器 → CNT

特点：
  → 专用外部触发引脚（如 TIM3 的 ETR = PD2）
  → 有独立的极性选择和预分频
  → 每个通用定时器都有 ETR 引脚
  → 真正的"外部"时钟源

使用场景：
  → 定时器级联（外部连接）
  → 对外部脉冲信号计数
  → 外部事件触发计数
```

### 3.5 内部触发（ITR）— 片内级联

```
信号路径：
  其他定时器 TRGO → ITR0~3 → 触发控制器 → CNT

可用信号：
  → ITR0, ITR1, ITR2, ITR3（共 4 个内部触发输入）
  → 对应不同定时器的触发输出

特点：
  → 不需要外部引脚，芯片内部连线
  → 每个定时器只能选择 4 个 ITR 中的一个
  → 信号来源有限，灵活性不如外部级联

使用场景：
  → 定时器内部级联（无需外部连线）
  → 一个定时器控制另一个定时器计数
  → 实现更长的定时时间
```

## 四、定时器级联

### 4.1 级联的概念

```
定时器级联：用一个定时器的输出控制另一个定时器的计数

目的：
  → 突破 16 位定时器的计数范围限制
  → 实现更长的定时时间
  → 多个定时器协同工作
```

### 4.2 级联方式

| 方式 | 连接 | 特点 |
|------|------|------|
| **外部引脚级联** | 定时器 A 输出 → GPIO 连线 → 定时器 B 的 ETR | 灵活，任意定时器可互连 |
| **外部通道级联** | 定时器 A 输出 → GPIO 连线 → 定时器 B 的 CH1/CH2 | 通过通道输入 |
| **内部级联** | 定时器 A 的 TRGO → 定时器 B 的 ITRx | **无需外部连线**，但只有 4 个 ITR 可选 |

### 4.3 级联示例

```
需求：实现超长定时（两个定时器级联）

TIM2（主定时器）：
  → 配置为产生周期性的 TRGO 输出
  → 每次 TIM2 溢出 → TRGO 输出一个脉冲

TIM3（从定时器）：
  → 时钟源选择 ITR（来自 TIM2）
  → 每收到一个 TRGO 脉冲 → CNT 加 1
  → TIM3 溢出时 → 产生最终的更新中断

总定时时间 = TIM2 定时时间 × TIM3 计数次数
例：TIM2 = 1s, TIM3 ARR = 999 → 总定时 = 1000s
```

## 五、计数模式（3 种）

### 5.1 模式对比

| 模式 | 计数方向 | 计数过程 | 溢出条件 |
|------|----------|----------|----------|
| **向上计数** | 0 → ARR | 递增 | CNT 超过 ARR |
| **向下计数** | ARR → 0 | 递减 | CNT 减到 0 以下 |
| **向上/向下交替** | 0 → ARR → 0 | 先增后减 | CNT 到达 ARR 或 0 |

### 5.2 向上计数（最常用）

```
CNT: 0 → 1 → 2 → ... → ARR → 溢出 → 0 → 1 → ...
                                     ↓
                              产生更新事件
```

### 5.3 向下计数

```
CNT: ARR → ARR-1 → ... → 1 → 0 → 溢出 → ARR → ...
                                        ↓
                                 产生更新事件
```

### 5.4 向上/向下交替（中央对齐）

```
CNT: 0 → 1 → ... → ARR → ARR-1 → ... → 1 → 0 → 1 → ...
                                  ↓
                           产生更新事件
特点：ARR 到达时和 0 到达时都会产生更新事件
用途：生成对称的 PWM 波形（电机控制常用）
```

## 六、输入捕获与输出比较（4 通道）

### 6.1 通道结构

```
每个通用定时器有 4 个通道（CH1~CH4）

每个通道包含：
  → 1 个捕获/比较寄存器（CCR）
  → 输入捕获电路
  → 输出比较电路

外部引脚复用：
  CH1 → 对应 GPIO 引脚（如 TIM3_CH1 = PA6）
  CH2 → 对应 GPIO 引脚（如 TIM3_CH2 = PA7）
  CH3 → 对应 GPIO 引脚（如 TIM3_CH3 = PB0）
  CH4 → 对应 GPIO 引脚（如 TIM3_CH4 = PB1）
```

### 6.2 输入捕获

```
功能：测量外部信号的脉宽、频率、占空比等

原理：
  外部信号 → CH 引脚 → 输入滤波 → 边沿检测 → 捕获 CCR
  → 信号边沿到来时，硬件自动将 CNT 值锁存到 CCR
  → 两次捕获的 CCR 差值 = 信号周期对应的计数值

使用场景：
  → 测量方波频率
  → 测量脉冲宽度
  → 超声波测距（测量回波时间）
```

### 6.3 输出比较

```
功能：根据 CNT 与 CCR 的比较结果控制输出电平

原理：
  CNT 持续递增 → 与 CCR 比较
  → CNT < CCR 时输出一种电平
  → CNT = CCR 时翻转/置位/清零输出
  → 产生特定的输出波形

使用场景：
  → 产生固定频率的方波
  → 产生 PWM 波形（特殊模式）
  → 定时翻转引脚电平
```

### 6.4 PWM 输出

```
PWM（Pulse Width Modulation，脉宽调制）

原理：
  ARR 决定 PWM 周期
  CCR 决定高电平持续时间（占空比）

  占空比 = CCR / (ARR + 1) × 100%

  CNT < CCR → 输出高电平
  CNT ≥ CCR → 输出低电平

使用场景：
  → 呼吸灯（LED 亮度控制）
  → 电机速度控制
  → 舵机角度控制
  → DAC 模拟输出
```

## 七、ETR 引脚对应关系

| 定时器 | ETR 引脚 | 其他复用功能 |
|--------|:--------:|-------------|
| TIM2 | PA0 | USART2_CTS, ADC12_IN0 |
| TIM3 | **PD2** | UART5_RX |
| TIM4 | PE0 | — |
| TIM5 | PA0 | 同 TIM2 |

> 通用定时器的 CH1~CH4 也有对应的 GPIO 引脚复用。

## 八、默认配置说明

```
不配置时钟源时：
  → 默认使用内部时钟 CK_INT = 72MHz

不配置计数模式时：
  → 默认向上计数（Up）

不配置 ARPE 时：
  → 默认关闭预装载（立即生效）
```

## 九、通用定时器功能总结

| 功能类别      | 具体功能                        | 说明            |
| --------- | --------------------------- | ------------- |
| **定时**    | 定时中断                        | 与基本定时器相同      |
| **计数模式**  | 向上 / 向下 / 向上向下交替            | 3 种计数模式       |
| **时钟源**   | 内部 / ETR / CH1/CH2 输入 / ITR | 4 种时钟来源       |
| **输入捕获**  | 测量信号频率、脉宽                   | 4 个通道         |
| **输出比较**  | 产生方波、翻转电平                   | 4 个通道         |
| **PWM**   | 脉宽调制输出                      | 输出比较的特殊模式     |
| **编码器**   | 增量编码器接口                     | 电机位置检测        |
| **霍尔传感器** | 霍尔传感器接口                     | 无刷电机控制        |
| **级联**    | 定时器之间级联                     | 突破计数范围限制      |
| **触发输出**  | TRGO                        | 触发其他外设（DAC 等） |
