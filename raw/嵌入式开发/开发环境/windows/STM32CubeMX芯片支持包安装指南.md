## 一句话定义
芯片支持包包含对应STM32系列的HAL库驱动、中间件等组件，是使用STM32CubeMX开发的必要前提。
## 核心内容
### 1. 支持包结构
- 基础包：如`stm32cube_fw_f1_v180.zip`，包含对应系列的核心驱动文件
- 扩展包：如`en.stm32cubef1-v1-8-5.zip`，提供bug修复和功能增强，依赖同版本基础包
### 2. 在线安装方法（推荐网络良好时使用）
1. 打开STM32CubeMX，进入`Help → Manage embedded software packages`
2. 找到对应STM32系列（如STM32F1），展开选择需要的版本
3. 点击"Install"按钮，等待下载安装完成（F1系列包约160MB）
4. 安装过程需要登录ST账号，无账号可免费注册
### 3. 离线安装方法（推荐网络受限环境使用）
#### 方案一：官方导入方式
1. 提前下载对应系列的基础包和扩展包
2. 进入包管理界面，点击"From Local"按钮，选择基础包导入安装
3. 扩展包需要手动复制到STM32Cube仓库目录，解压覆盖基础包文件
#### 方案二：手动覆盖方式
1. 将基础包`en.stm32cubef1.zip`改名为`stm32cube_fw_f1_v180.zip`
2. 复制到仓库目录后解压
3. 再将扩展包解压覆盖到对应目录
4. 重启STM32CubeMX即可识别到安装的支持包
### 4. 版本选择建议
- 优先选择最新稳定版本，如当前F1系列推荐v1.8.5
- 避免使用预览版或测试版，保证稳定性
## 注意事项&踩坑
1. 扩展包安装必须依赖对应版本的基础包，版本不匹配会安装失败
2. 离线安装时文件名必须严格匹配要求，否则无法识别
3. 安装前建议备份原有支持包，避免升级失败影响已有项目
4. 下载速度慢时可优先使用离线安装方式，避免在线下载中断
