# 脚本目录说明

> 最后更新：2026-06-02

## 脚本清单

### generate_index.py
- **用途**：自动扫描 raw/ 和 core/ 目录，生成 index/全文目录.md
- **使用方式**：`python generate_index.py`
- **输入**：raw/ 和 core/ 文件夹
- **输出**：index/全文目录.md
- **依赖**：Python 3.6+
- **状态**：✅ 可用
- **最后更新**：2026-06-02
- **备注**：摘要提取自动跳过表格、代码块等非正文内容

### check_status.py
- **用途**：检查 raw/ 文件的提炼状态，显示已提炼和待提炼文件
- **使用方式**：`python check_status.py`
- **输入**：raw/ 和 core/ 文件夹
- **输出**：终端状态报告
- **依赖**：Python 3.6+
- **状态**：✅ 可用
- **最后更新**：2026-06-02
- **备注**：检查 core/ 文件的来源清单判断提炼状态
