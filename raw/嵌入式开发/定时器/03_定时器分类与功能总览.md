## 一、定时器分类总览

| 类型 | 定时器 | 通道 | 特有功能 | 总线 | 时钟 |
|------|--------|:----:|----------|------|:----:|
| **基本** | TIM6/7 | 0 | RCR | APB1 | 72MHz |
| **通用** | TIM2~5 | 4 | 输入捕获/输出比较/从模式 | APB1 | 72MHz |
| **高级** | TIM1/8 | 4 | 互补输出/死区/刹车/RCR | APB2 | 72MHz |

## 二、寄存器配置方式总结

### 2.1 输出比较 / PWM 输出

```
关键寄存器：CCMR1（CCxS=00, OCxM=110）+ CCER（CCxP, CCxE）
GPIO：复用推挽输出（CNF=10, MODE=11）
```

### 2.2 输入捕获（测周期）

```
关键寄存器：CCMR1（CC1S=01, IC1F, IC1PSC）+ CCER（CC1P=0, CC1E=1）
GPIO：复用浮空输入（CNF=10, MODE=00）
中断：DIER.CC1IE=1 + NVIC
```

### 2.3 PWM 输入模式（测周期+占空比）

```
关键寄存器：
  CCMR1（CC1S=01直通, CC2S=10交叉）
  CCER（CC1P=0上升沿, CC2P=1下降沿, CC1E=1, CC2E=1）
  SMCR（TS=101, SMS=100）
不需要中断，硬件自动复位
```

## 三、HAL 库方式总结

### 3.1 启动函数选择

| 场景 | 函数 |
|------|------|
| PWM 输出 | `HAL_TIM_PWM_Start()` |
| 输入捕获 | `HAL_TIM_IC_Start()` |
| 输入捕获+中断 | `HAL_TIM_IC_Start_IT()` |
| 高级定时器 PWM+更新中断 | `HAL_TIM_PWM_Start()` + `__HAL_TIM_ENABLE_IT(TIM_IT_UPDATE)` |

### 3.2 读写 CCR

```c
__HAL_TIM_SET_COMPARE(&htim, channel, value);   // 写
__HAL_TIM_GET_COMPARE(&htim, channel);           // 读
```

## 五、关键概念

| 概念           | 说明                            |
| ------------ | ----------------------------- |
| **影子寄存器**    | 真正工作的寄存器，更新事件时从预装载刷新          |
| **更新事件**     | CNT 溢出产生，刷新影子寄存器              |
| **事件 vs 中断** | 事件=硬件瞬间完成，中断=软件延迟执行           |
| **从模式复位**    | TRGI 上升沿→CNT 自动清零，无需中断        |
| **RCR**      | 高级定时器重复计数器，闪烁次数=RCR+1         |
| **MOE**      | 高级定时器主输出使能（总开关）               |
| **直通 vs 交叉** | CCxS=01 直通本通道，CCxS=10 交叉到另一通道 |
