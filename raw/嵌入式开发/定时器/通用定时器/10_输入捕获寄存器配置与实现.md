## 一、实验硬件连接

### 1.1 信号源与捕获端

| 设备 | 定时器 | 通道 | 引脚 | 功能 |
|------|--------|:----:|:----:|------|
| **信号源** | TIM5 | CH2 | **PA1** | PWM 输出（呼吸灯时的信号） |
| **捕获端** | TIM4 | CH1 | **PB6** | 输入捕获（测量 PWM 周期） |

### 1.2 硬件连接方式

```
PA1 (TIM5_CH2, PWM 输出) ──硬连线──→ PB6 (TIM4_CH1, 输入捕获)

  ┌──────┐    外部导线     ┌──────┐
  │ TIM5 │────────────────→│ TIM4 │
  │ CH2  │   PA1 → PB6    │ CH1  │
  │ PWM  │                 │ 捕获 │
  └──────┘                 └──────┘
```

### 1.3 PB6 的复用说明

| 复用功能 | 说明 |
|----------|------|
| TIM4_CH1 | 本实验使用（输入捕获） |
| I2C1_SCL | 不使用 |

## 二、涉及的寄存器总览

| 寄存器 | 功能 | 本实验操作 |
|--------|------|-----------|
| **CR1** | 控制寄存器 1 | CEN（使能计数器） |
| **CCMR1** | 捕获/比较模式寄存器 1 | CC1S=01（输入模式）、IC1F（滤波）、IC1PSC（预分频） |
| **CCER** | 捕获/比较使能寄存器 | CC1P（边沿选择）、CC1E（使能） |
| **CCR1** | 捕获/比较寄存器 1 | 存储捕获到的计数器值 |
| **DIER** | DMA/中断使能寄存器 | CC1IE（捕获中断使能） |
| **SR** | 状态寄存器 | CC1IF（捕获中断标志） |
| **PSC** | 预分频器 | 计数器时钟分频 |
| **ARR** | 自动重装载寄存器 | 设为最大值 65535 |

## 三、CCMR1 — 输入模式配置

### 3.1 通道 1 输入模式位定义（低 8 位）

| 位 | 名称 | 功能 | 本实验 |
|:--:|------|------|:------:|
| 1:0 | **CC1S** | 捕获/比较选择 | **01** = 输入（映射到 TI1） |
| 3:2 | **IC1PSC** | 输入预分频 | **00** = 不分频 |
| 7:4 | **IC1F** | 输入滤波器 | **0000** = 不滤波 |

### 3.2 CC1S — 输入映射选择

```
CC1S = 00 → 输出比较模式
CC1S = 01 → 输入模式，IC1 映射到 TI1（直通）    ← 本实验
CC1S = 10 → 输入模式，IC1 映射到 TI2（交叉映射）
CC1S = 11 → 输入模式，IC1 映射到 TRC（触发输入）
```

### 3.3 CC1S 映射关系图

```
CC1S = 01（直通）：
  CH1 引脚 → TI1 → 滤波 → 边沿检测 → TI1FP1 → 多路选择 → IC1 → CCR1
                                        ↑
                                   直通选择

CC1S = 10（交叉）：
  CH2 引脚 → TI2 → 滤波 → 边沿检测 → TI2FP1 → 多路选择 → IC1 → CCR1
                                        ↑
                                   交叉选择

CC1S = 11（TRC）：
  触发输入 → TRC → 多路选择 → IC1 → CCR1
```

### 3.4 IC1PSC — 输入预分频

| 值 | 分频 | 说明 |
|:--:|:----:|------|
| 00 | 不分频 | 每个事件都触发捕获 |
| 01 | 2 分频 | 每 2 个事件触发一次 |
| 10 | 4 分频 | 每 4 个事件触发一次 |
| 11 | 8 分频 | 每 8 个事件触发一次 |

### 3.5 IC1F — 输入滤波器

```
4 位滤波器配置，决定采样频率和滤波效果

IC1F = 0000：不滤波（信号质量好时使用）
IC1F ≠ 0000：根据设定频率进行数字滤波（滤除毛刺）

本实验：信号质量好，不滤波 → IC1F = 0000
```

## 四、CCER — 边沿选择与使能

### 4.1 通道 1 关键位

| 位 | 名称 | 功能 | 本实验 |
|:--:|------|------|:------:|
| 0 | **CC1E** | 捕获/比较使能 | **1** = 使能 |
| 1 | **CC1P** | 输入极性（边沿选择） | **0** = 上升沿捕获 |

### 4.2 CC1P 在输入模式下的含义

```
CC1P = 0：捕获上升沿  ← 本实验
CC1P = 1：捕获下降沿

⚠️ 输入模式与输出模式的 CCxP 含义完全不同：
  输入模式：选择捕获哪种边沿
  输出模式：选择有效电平（高/低）
```

### 4.3 CC1E — 使能

```
CC1E = 0：捕获功能禁用
CC1E = 1：捕获功能使能（必须开启）
```

## 五、DIER — 捕获中断使能

### 5.1 关键位

| 位 | 名称 | 功能 | 本实验 |
|:--:|------|------|:------:|
| 1 | **CC1IE** | 捕获/比较 1 中断使能 | **1** = 使能 |

### 5.2 与基本定时器的 DIER 对比

```
基本定时器 DIER：
  仅有 UIE（更新中断使能）

通用定时器 DIER：
  UIE  — 更新中断使能
  CC1IE — 通道 1 捕获/比较中断使能
  CC2IE — 通道 2 捕获/比较中断使能
  CC3IE — 通道 3 捕获/比较中断使能
  CC4IE — 通道 4 捕获/比较中断使能
  ...（更多位，通用/高级定时器功能更多）
```

## 六、SR — 捕获中断标志

### 6.1 关键位

| 位 | 名称 | 功能 | 操作 |
|:--:|------|------|------|
| 1 | **CC1IF** | 捕获/比较 1 中断标志 | **软件写 0 清除** |

### 6.2 置位与清除

```
置位条件：
  → 捕获事件发生时（上升沿/下降沿到来）
  → CNT 值已被锁存到 CCR
  → 硬件自动置位

清除方式：
  → 软件写 0 清除
  → 读 CCR 寄存器也可能清除（取决于配置）
```

---

## 七、CCR1 — 捕获值存储

```
捕获事件发生时：
  → 硬件自动将 CNT 的当前值复制到 CCR1
  → 在下次捕获之前，CCR1 值保持不变
  → 读取 CCR1 即可获得捕获时刻的计数器值

测量周期：
  周期 = CCR1(第二次捕获) - CCR1(第一次捕获)
  单位 = 计数器时钟周期（本实验为 1μs）
```

---

## 八、定时器参数配置

### 8.1 计数器时钟设置

```
目标：每次计数 = 1μs

PSC = 71 → 分频值 = 72
计数器时钟 = 72MHz / 72 = 1MHz = 1μs/次
```

### 8.2 ARR 设置

```
ARR = 65535（16 位最大值）

目的：避免在测量一个周期内溢出
最大可测周期 = 65535μs ≈ 65.5ms
对应最小频率 = 1/65.5ms ≈ 15.26Hz
```

### 8.3 测量范围

| 参数 | 值 | 说明 |
|------|:--:|------|
| 计数器时钟 | 1MHz | 1μs/次 |
| 最小可测周期 | 1μs | 信号频率上限 1MHz |
| 最大可测周期 | 65535μs | 信号频率下限 15.26Hz |
| 测量精度 | 1μs | 由计数器时钟决定 |

## 九、完整配置流程

```
① 开启时钟
   ├── RCC->APB1ENR |= TIM4EN     → TIM4 时钟
   └── RCC->APB2ENR |= IOPBEN     → GPIOB 时钟

② 配置 GPIO（PB6 复用浮空输入）
   → CNF = 10（复用功能）
   → MODE = 00（输入模式）

③ 配置 PSC = 71（72 分频 → 1MHz）

④ 配置 ARR = 65535（最大值）

⑤ 配置 CCMR1（输入模式）
   ├── CC1S = 01（输入，映射到 TI1）
   ├── IC1PSC = 00（不分频）
   └── IC1F = 0000（不滤波）

⑥ 配置 CCER
   ├── CC1P = 0（上升沿捕获）
   └── CC1E = 1（使能捕获）

⑦ 配置 DIER
   └── CC1IE = 1（使能捕获中断）

⑧ 配置 NVIC
   ├── NVIC_SetPriorityGrouping(3)
   ├── NVIC_SetPriority(TIM4_IRQn, 2)
   └── NVIC_EnableIRQ(TIM4_IRQn)

⑨ 使能定时器
   └── TIM4->CR1 |= CEN
```

### 10.2 tim4.c

```c
#include "tim4.h"

volatile uint32_t capture_value = 0;      // 捕获到的周期值
volatile uint8_t  capture_flag  = 0;      // 捕获完成标志

static uint32_t last_capture = 0;         // 上一次捕获值

void TIM4_IC_Init(void)
{
    /* ① 开启时钟 */
    RCC->APB1ENR |= RCC_APB1ENR_TIM4EN;
    RCC->APB2ENR |= RCC_APB2ENR_IOPBEN;

    /* ② GPIO：PB6 复用浮空输入 */
    GPIOB->CRL &= ~GPIO_CRL_MODE6;       // MODE = 00（输入）
    GPIOB->CRL |=  GPIO_CRL_CNF6_1;      // CNF = 10（复用输入）
    GPIOB->CRL &= ~GPIO_CRL_CNF6_0;      //

    /* ③ PSC：72 分频 → 1MHz（1μs/次） */
    TIM4->PSC = 71;

    /* ④ ARR：最大值，避免溢出 */
    TIM4->ARR = 65535;

    /* ⑤ CCMR1：输入模式配置 */
    // CC1S = 01（输入，映射到 TI1）
    TIM4->CCMR1 |=  TIM_CCMR1_CC1S_0;
    TIM4->CCMR1 &= ~TIM_CCMR1_CC1S_1;
    // IC1PSC = 00（不分频）
    TIM4->CCMR1 &= ~TIM_CCMR1_IC1PSC;
    // IC1F = 0000（不滤波）
    TIM4->CCMR1 &= ~TIM_CCMR1_IC1F;

    /* ⑥ CCER：上升沿捕获 + 使能 */
    TIM4->CCER &= ~TIM_CCER_CC1P;        // CC1P = 0（上升沿）
    TIM4->CCER |=  TIM_CCER_CC1E;        // CC1E = 1（使能）

    /* ⑦ DIER：使能捕获中断 */
    TIM4->DIER |= TIM_DIER_CC1IE;

    /* ⑧ NVIC 配置 */
    NVIC_SetPriorityGrouping(3);
    NVIC_SetPriority(TIM4_IRQn, 2);
    NVIC_EnableIRQ(TIM4_IRQn);

    /* ⑨ 使能定时器 */
    TIM4->CR1 |= TIM_CR1_CEN;
}

/* 中断服务函数 */
void TIM4_IRQHandler(void)
{
    if (TIM4->SR & TIM_SR_CC1IF)
    {
        uint32_t current_capture = TIM4->CCR1;    // 读取捕获值（自动清除标志）

        if (last_capture != 0)
        {
            // 计算周期（处理溢出情况）
            if (current_capture >= last_capture)
                capture_value = current_capture - last_capture;
            else
                capture_value = 65536 - last_capture + current_capture;

            capture_flag = 1;    // 标记捕获完成
        }

        last_capture = current_capture;

        TIM4->SR &= ~TIM_SR_CC1IF;    // 清除中断标志
    }
}
```

### 10.3 main.c

```c
#include "stm32f10x.h"
#include "tim4.h"
#include "tim5.h"
#include "delay.h"
#include <stdio.h>

extern volatile uint32_t capture_value;
extern volatile uint8_t  capture_flag;

int main(void)
{
    /* 初始化 TIM5 输出 PWM */
    TIM5_Init();
    TIM5_Start();
    TIM5_SetDutyCycle(50);    // 50% 占空比

    /* 初始化 TIM4 输入捕获 */
    TIM4_IC_Init();

    printf("Input Capture Test\n");

    while (1)
    {
        if (capture_flag)
        {
            capture_flag = 0;

            float frequency = 1000000.0f / capture_value;    // Hz
            float period_ms = capture_value / 1000.0f;        // ms

            printf("Period: %.2f ms, Freq: %.2f Hz\n", period_ms, frequency);
        }

        Delay_ms(500);
    }
}
```

## 十一、溢出处理

### 11.1 溢出场景

```
如果信号周期 > 65535μs（频率 < 15.26Hz）：
  → CNT 溢出归零后再捕获
  → current_capture < last_capture
  → 需要特殊处理

处理方式：
  周期 = 65536 - last_capture + current_capture
```

### 11.2 代码中的溢出判断

```c
if (current_capture >= last_capture)
    capture_value = current_capture - last_capture;      // 正常
else
    capture_value = 65536 - last_capture + current_capture;  // 溢出
```

| ARR | 99（100 级） | 65535（最大值） |
| CCR | CCR2（占空比） | CCR1（捕获值） |
| NVIC | 不需要 | 需要配置 |

## 十六、测量结果计算

```
捕获到的 capture_value = 两次上升沿之间的计数值

周期 T = capture_value × 1μs
       = capture_value μs

频率 f = 1 / T
       = 1,000,000 / capture_value Hz
       = 1,000 / capture_value kHz

示例：
  TIM5 PWM 频率 = 100Hz → 周期 = 10ms = 10000μs
  捕获值应为 capture_value ≈ 10000
  验证：f = 1,000,000 / 10000 = 100Hz ✅
```

    │
