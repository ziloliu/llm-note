# 通用定时器触发输入与从模式详解 笔记

---

## 一、测量占空比的需求

### 1.1 目标

```
PWM 方波三个核心参数：
  ① 周期 T（两个连续上升沿之间的时间）
  ② 频率 f = 1/T
  ③ 占空比 D = 高电平时间 t / 周期 T

已实现：周期和频率的测量
待实现：占空比的测量
```

### 1.2 占空比测量原理

```
信号：  ┌─────┐              ┌─────┐
        │     │              │     │
   ─────┘     └──────────────┘     └─────
        ↑     ↑              ↑
       上升沿  下降沿         上升沿
        |← t →|
        |←──────── T ────────→|

占空比 D = t / T

需要：
  → 两个连续上升沿之间的时间 = 周期 T
  → 上升沿到紧接着的下降沿之间的时间 = 高电平时间 t
```

### 1.3 为什么之前的方案不够用

| 方案 | 能否测占空比 | 原因 |
|------|:------------:|------|
| 只捕获上升沿 | ❌ | 只能测周期，测不了高电平时间 |
| 只捕获下降沿 | ❌ | 只能测周期，测不了高电平时间 |
| 改极性切换 | ❌ | 同一时刻只能选一种边沿 |
| 单通道单边沿 | ❌ | 没有"同时检测两种边沿"的配置 |

> 核心矛盾：CCER.CCxP 只能选择一种边沿，无法同时检测上升沿和下降沿。

## 二、触发信号概述

### 2.1 触发控制器在框图中的位置

```
┌─────────────────────────────────────────────────┐
│                 通用定时器                        │
│                                                 │
│  ┌──────────────────────────────────┐           │
│  │        触发控制器                  │           │
│  │                                  │           │
│  │  多路选择 ──→ TRGI ──→ 控制逻辑   │           │
│  │  (8选1)     (触发输入)  ↓         │           │
│  │              复位/使能/计数控制    │           │
│  │                                  │           │
│  │              TRGO ──→ 输出        │           │
│  │           (触发输出)              │           │
│  └──────────────────────────────────┘           │
│         ↕                    ↕                   │
│   输入通道               输出通道                 │
└─────────────────────────────────────────────────┘
```

### 2.2 两类触发信号

| 信号 | 全称 | 方向 | 功能 |
|------|------|------|------|
| **TRGI** | Trigger Input | **输入** | 接收外部信号，控制本定时器行为 |
| **TRGO** | Trigger Output | **输出** | 本定时器产生信号，驱动其他定时器/外设 |

## 三、主模式与从模式

### 3.1 定义

| 模式 | 定义 | 信号方向 | 角色 |
|------|------|----------|------|
| **主模式** | 本定时器的 TRGO 输出给其他定时器 | 输出驱动别人 | 主动，发出指令 |
| **从模式** | 其他定时器的 TRGO 接到本定时器的 TRGI | 被别人驱动 | 被动，接收指令 |

### 3.2 级联关系

```
定时器 A（主）                    定时器 B（从）
    │                                │
    ├── TRGO ──────────→ TRGI ──────┤
    │                                │
    │ 发出控制信号                     │ 接收控制信号
    │ 驱动别人                        │ 被别人驱动

例：TIM1.TRGO → TIM2.ITR0
  → TIM1 是主，TIM2 是从
  → TIM1 的输出控制 TIM2 的行为
```

## 四、TRGI 触发输入信号来源

### 4.1 SMCR.TS 触发选择（3 位，8 种选择）

| TS[2:0] | 类型 | 触发源 | 说明 |
|:-------:|------|--------|------|
| **000** | 内部触发 | ITR0 | 来自其他定时器（内部连线） |
| **001** | 内部触发 | ITR1 | 来自其他定时器（内部连线） |
| **010** | 内部触发 | ITR2 | 来自其他定时器（内部连线） |
| **011** | 内部触发 | ITR3 | 来自其他定时器（内部连线） |
| **100** | TI1 边沿检测 | TI1F_ED | 通道 1 上升沿+下降沿都检测 |
| **101** | TI1 直连 | TI1FP1 | 通道 1 经滤波极性处理（单一边沿） |
| **110** | TI2 直连 | TI2FP2 | 通道 2 经滤波极性处理（单一边沿） |
| **111** | 外部触发 | ETRF | 外部触发引脚输入 |

### 4.2 第一类：内部触发（ITR0~ITR3）

```
来自芯片内部其他定时器的 TRGO 信号

内部连线是固定的，不可更改：

  从定时器 ┃ ITR0     ITR1     ITR2     ITR3
  ─────────╋────────────────────────────────
  TIM2     ┃ TIM1     TIM8     TIM3     TIM4
  TIM3     ┃ TIM1     TIM2     TIM5     TIM4
  TIM4     ┃ TIM1     TIM2     TIM3     TIM8
  TIM5     ┃ TIM2     TIM3     TIM4     TIM8

例：TIM2 选 ITR0 → 触发源是 TIM1 的 TRGO
    TIM5 选 ITR0 → 触发源是 TIM2 的 TRGO
```

### 4.3 第二类：外部触发（ETRF）

```
来自外部引脚 ETR 的信号

路径：
  ETR 引脚 → 极性选择 → 边沿检测 → 预分频 → ETRF → 多路选择 → TRGI

特点：
  → 由外部硬件信号触发
  → 可以接外部设备的信号
```

### 4.4 第三类：TI1 边沿检测（TI1F_ED）

```
来源：通道 1 输入信号
路径：CH1 引脚 → TI1 → 滤波 → 双边沿检测 → TI1F_ED

特点：
  → 上升沿和下降沿都能检测
  → ED = Edge Detect（边沿检测）
  → 任何一个边沿都会产生触发信号
  → 不受 CCER.CCxP 极性选择影响（双边沿都有效）
```

### 4.5 第四类：TI1/TI2 直连（TI1FP1 / TI2FP2）

```
来源：通道 1 或通道 2 的输入信号
路径：CHx 引脚 → TIx → 滤波 → 单边沿检测(极性选择) → TIxFPx

特点：
  → 只检测一种边沿（由 CCER.CCxP 决定）
  → FP = Filtered + Polarity（滤波+极性）
  → 与输入捕获共用相同的滤波和极性配置

本实验使用的方案：
  TI1FP1 → 选择上升沿 → 控制计数器复位
  TI2FP2 → 选择下降沿 → 用于捕获高电平结束时刻
```

## 五、从模式（SMCR.SMS）

### 5.1 SMCR.SMS 从模式选择（3 位，8 种模式）

| SMS[2:0] | 模式 | 功能 |
|:--------:|------|------|
| **000** | **关闭从模式** | 默认，使用内部时钟，TRGI 无效 |
| 001 | 编码器模式 1 | TI1FP1 计数，TI2FP2 边沿计数 |
| 010 | 编码器模式 2 | TI2FP2 计数，TI1FP1 边沿计数 |
| 011 | 编码器模式 3 | TI1FP1 和 TI2FP2 都计数 |
| **100** | **复位模式** | TRGI 上升沿 → CNT 复位（清零） |
| 101 | 门控模式 | TRGI 高电平 → 计数器使能；低电平 → 停止 |
| 110 | 触发模式 | TRGI 上升沿 → 启动计数器（不清零） |
| **111** | **外部时钟模式** | TRGI 上升沿 → 驱动 CNT 加 1（替代内部时钟） |

### 5.2 各模式详解

#### 000 — 关闭从模式（默认）

```
→ TRGI 信号被忽略
→ 计数器由内部时钟驱动
→ 普通定时器功能
```

#### 001/010/011 — 编码器模式

```
→ 用于连接旋转编码器
→ TI1FP1 和 TI2FP2 信号控制计数方向和计数
→ 三种子模式对应不同的编码器接口方式
→ 本实验不使用
```

#### 100 — 复位模式（本实验核心）

```
→ TRGI 信号的上升沿 → 将 CNT 复位为 0
→ 同时产生更新事件（重装载寄存器重新加载）

应用场景：
  → 测量 PWM 周期（自动清零计数器，无需中断）
  → 测量 PWM 占空比（配合双通道捕获）

触发信号来源（本实验）：
  → TI1FP1：上升沿，连接到复位模式
  → 每次 PWM 上升沿到来 → CNT 自动清零
```

#### 101 — 门控模式

```
→ TRGI 高电平 → 计数器使能（开始计数）
→ TRGI 低电平 → 计数器停止（暂停，不清零）

类似软件 CEN 使能位，但由硬件信号控制
暂停后恢复可继续计数（不会重新开始）
```

#### 110 — 触发模式

```
→ TRGI 上升沿 → 启动计数器
→ 不复位，只是启动
→ 没有自动停止功能
```

#### 111 — 外部时钟模式

```
→ TRGI 上升沿 → 驱动 CNT 加 1
→ 替代内部时钟成为计数器的时钟源
→ 这是唯一改变计数时钟源的模式

⚠️ 其他模式的计数时钟仍然是内部时钟！
   只有 111 模式改变了时钟源
```

### 5.3 各模式对计数器的影响

| 模式 | CNT 时钟 | 复位 | 使能控制 | 计数方向 |
|:----:|:--------:|:----:|:--------:|:--------:|
| 000 | 内部时钟 | — | — | — |
| 100 复位 | 内部时钟 | ✅ TRGI 上升沿清零 | — | — |
| 101 门控 | 内部时钟 | — | ✅ TRGI 高/低 | — |
| 110 触发 | 内部时钟 | — | ✅ TRGI 上升沿启动 | — |
| 111 外部时钟 | **TRGI** | — | — | 向上 |

## 六、测量占空比的实现思路

### 6.1 核心思想

```
使用从模式的复位模式 + 双通道输入捕获

信号输入：
  CH1 → TI1FP1（上升沿检测）→ 连接到 TRGI → 控制 CNT 复位
  CH2 → TI2FP2（下降沿检测）→ 捕获高电平结束时刻的 CNT 值

流程：
  ① 上升沿到来 → TRGI 触发 → CNT 自动清零（硬件复位）
  ② CNT 从 0 开始递增
  ③ 下降沿到来 → CCR2 捕获当前 CNT 值 = 高电平时间 t
  ④ 下一个上升沿到来 → CCR1 捕获 CNT 值 = 周期 T（同时 CNT 复位）
  ⑤ 占空比 D = CCR2 / CCR1 = t / T
```

### 6.2 时序图

```
PWM 信号：
      ┌─────┐              ┌─────┐
      │     │              │     │
 ─────┘     └──────────────┘     └─────
      ↑     ↑              ↑
     ①    ②             ③

① 上升沿：
  → TRGI 触发复位模式 → CNT = 0
  → CCR1 = CNT（捕获到 0 或上次周期值）

② 下降沿：
  → CCR2 = CNT = t（高电平时间，如 6000μs）

③ 下一个上升沿：
  → CCR1 = CNT = T（完整周期，如 10000μs）
  → TRGI 触发复位模式 → CNT = 0（自动清零）

占空比 = CCR2 / CCR1 = 6000 / 10000 = 60%
```

### 6.3 为什么需要从模式

| 没有从模式 | 有从模式（复位模式） |
|:----------:|:-------------------:|
| 需要中断清零 CNT | 硬件自动清零 CNT |
| 中断响应有延迟 | 硬件瞬间完成 |
| 需要状态机（count 变量） | 无需任何状态管理 |
| 只能测一种边沿时间 | 可同时测周期和高电平时间 |

## 七、SMCR 寄存器结构

```
SMCR（16位）：

bit[2:0]   SMS[2:0]    从模式选择（8种模式）
bit[6:4]   —           保留
bit[7]     ETP         外部触发极性
bit[8]     ECE         外部时钟使能
bit[10:9]  ETPS[1:0]   外部触发预分频
bit[14:11] ETF[3:0]    外部触发滤波
bit[15]    MSM         主/从模式

本实验关键配置：
  SMS = 100（复位模式）
  TS = 101（TI1FP1 作为触发源）
```

## 八、测量占空比完整方案

### 8.1 定时器通道分配

| 通道 | 信号 | 极性 | 用途 |
|:----:|------|------|------|
| CH1 (PB6) | TI1FP1 | 上升沿 | ① 连接 TRGI 复位 CNT ② CCR1 捕获周期 T |
| CH2 (PB7) | TI2FP2 | 下降沿 | CCR2 捕获高电平时间 t |

### 8.2 SMCR 配置

| 位 | 值 | 说明 |
|:--:|:--:|------|
| SMS | **100** | 复位模式 |
| TS | **101** | TI1FP1 作为触发输入 |

### 8.3 CCMR 配置

| 寄存器 | 通道 | 位 | 值 | 说明 |
|--------|:----:|:--:|:--:|------|
| CCMR1 | CH1 | CC1S | 01 | 输入模式，映射到 TI1 |
| CCMR1 | CH2 | CC2S | 01 | 输入模式，映射到 TI2 |

### 8.4 CCER 配置

| 位 | 值 | 说明 |
|:--:|:--:|------|
| CC1P | **0** | CH1 上升沿捕获 |
| CC1E | 1 | CH1 捕获使能 |
| CC2P | **1** | CH2 下降沿捕获 |
| CC2E | 1 | CH2 捕获使能 |

### 8.5 信号流图

```
PB6 (CH1) ──→ TI1 ──→ 滤波 ──→ 上升沿检测 ──→ TI1FP1
                                                    │
                    ┌───────────────────────────────┤
                    ↓                               ↓
              触发选择(TS=101)              IC1 捕获(CCR1)
                    ↓
              复位模式(SMS=100)
                    ↓
              CNT = 0（自动清零）

PB7 (CH2) ──→ TI2 ──→ 滤波 ──→ 下降沿检测 ──→ TI2FP2
                                                    ↓
                                            IC2 捕获(CCR2)
```

## 九、从模式与中断方式对比

| 对比项 | 中断方式（之前） | 从模式复位方式 |
|--------|:---------------:|:-------------:|
| CNT 清零 | 中断服务函数中软件清零 | 硬件自动清零 |
| 响应速度 | 延迟若干时钟周期 | **瞬间完成** |
| 代码复杂度 | 需要中断函数 | **无需中断清零代码** |
| 测量周期 | ✅ | ✅ |
| 测量高电平 | ❌ | ✅ |
| 测量占空比 | ❌ | ✅ |
| 全局变量 | 需要 count/cycle | **不需要** |

## 十、关键概念总结

| 概念 | 说明 |
|------|------|
| **TRGI** | 触发输入信号，从外部或其他定时器接收 |
| **TRGO** | 触发输出信号，输出给其他定时器或外设 |
| **主模式** | 本定时器的 TRGO 驱动其他定时器 |
| **从模式** | 其他信号通过 TRGI 控制本定时器行为 |
| **SMCR.TS** | 触发源选择（8 种来源） |
| **SMCR.SMS** | 从模式选择（8 种模式） |
| **复位模式 (100)** | TRGI 上升沿自动清零 CNT |
| **TI1FP1** | 通道 1 输入经滤波极性处理后的信号 |
| **TI2FP2** | 通道 2 输入经滤波极性处理后的信号 |
| **TI1F_ED** | 通道 1 双边沿检测信号 |
| **级联** | 定时器之间通过 TRGO/TRGI 连接，形成主从关系 |
| **ITR0~3** | 内部触发连接线，连接关系固定不可更改 |
