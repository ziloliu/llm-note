## 二、主函数代码

### 2.1 头文件引入

```c
#include "main.h"
#include "usart.h"
#include "m24c02.h"
#include <stdio.h>
```

### 2.2 完整测试代码

```c
int main(void)
{
    /* ① 初始化 */
    HAL_Init();
    SystemClock_Config();
    MX_GPIO_Init();
    MX_USART1_UART_Init();
    EEPROM_Init();

    printf("I2C 软件模拟实验开始\n");

    /* ② 单字节写入 */
    EEPROM_WriteByte(0x00, 'A');
    EEPROM_WriteByte(0x01, 'B');
    EEPROM_WriteByte(0x02, 'C');

    /* ③ 单字节读取 */
    uint8_t byte1 = EEPROM_ReadByte(0x00);
    uint8_t byte2 = EEPROM_ReadByte(0x01);
    uint8_t byte3 = EEPROM_ReadByte(0x02);

    /* ④ 串口打印（单字节） */
    printf("byte1=%c\tbyte2=%c\tbyte3=%c\n", byte1, byte2, byte3);

    /* ⑤ 多字节写入 */
    EEPROM_WriteBytes(0x00, "123456", 6);

    /* ⑥ 多字节读取 */
    uint8_t buffer[100] = {0};
    EEPROM_ReadBytes(0x00, buffer, 6);

    /* ⑦ 串口打印（多字节） */
    printf("buffer=%s\n", buffer);

    /* ⑧ 超 16 字节写入测试 */
    memset(buffer, 0, sizeof(buffer));   // 清零缓冲区
    EEPROM_WriteBytes(0x00,
        "1234567890abcdefjhijk", 21);    // 21 字节，超过页大小(16)
    EEPROM_ReadBytes(0x00, buffer, 21);
    printf("buffer=%s\n", buffer);

    while (1)
    {
    }
}
```

## 四、页写绕回现象分析

### 4.1 写入 21 字节的实际行为

```
AT24C02 第 0 页（地址 0x00 ~ 0x0F，共 16 字节）：

写入顺序：
  第 1 字节：'1'  → 地址 0x00
  第 2 字节：'2'  → 地址 0x01
  ...
  第16 字节：'f'  → 地址 0x0F
  ─────────── 页写满，地址绕回 ───────────
  第17 字节：'j'  → 地址 0x00（覆盖 '1'）
  第18 字节：'h'  → 地址 0x01（覆盖 '2'）
  第19 字节：'i'  → 地址 0x02（覆盖 '3'）
  第20 字节：'j'  → 地址 0x03（覆盖 '4'）
  第21 字节：'k'  → 地址 0x04（覆盖 '5'）

最终内存状态：
  地址 0x00~0x04：j h i j k（被覆盖）
  地址 0x05~0x0F：6 7 8 9 0 a b c d e f（原数据保留）
```

### 4.2 验证

```
读出 21 字节：
  buffer = "jhijk7890abcdef" + 5个 '\0'

与预期一致 ✓
```

### 4.3 页绕回示意

```
第 0 页内存布局（16 字节）：

写入 "1234567890abcdefjhijk"（21 字节）：

第一轮（字节 1~16）：
┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │ 7 │ 8 │ 9 │ 0 │ a │ b │ c │ d │ e │ f │
└───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┘
 0x00                                                        0x0F

第二轮（字节 17~21，绕回覆盖）：
┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┐
│ j │ h │ i │ j │ k │ 6 │ 7 │ 8 │ 9 │ 0 │ a │ b │ c │ d │ e │ f │
└───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┘

```

## 五、memset 函数说明

```c
memset(buffer, 0, sizeof(buffer));
```

| 参数 | 含义 |
|------|------|
| `buffer` | 目标内存首地址 |
| `0` | 要设置的值 |
| `sizeof(buffer)` | 要设置的字节数 |

> 来自 `<string.h>`，将 buffer 全部清零，确保后续 `printf("%s")` 输出不会因残留数据而显示乱码。

## 六、跨页写入正确做法

```c
// 需求：从地址 0x0E 开始写入 21 字节

// 第一次：写 2 字节到第 0 页末尾（0x0E~0x0F）
EEPROM_WriteBytes(0x0E, data, 2);

// 第二次：写 16 字节到第 1 页（0x10~0x1F）
EEPROM_WriteBytes(0x10, data + 2, 16);

// 第三次：写 3 字节到第 2 页开头（0x20~0x22）
EEPROM_WriteBytes(0x20, data + 18, 3);
```
