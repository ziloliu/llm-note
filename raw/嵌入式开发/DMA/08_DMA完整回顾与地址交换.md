## 一、DMA 与中断的系统定位

### 1.1 两者共同点

| 特性 | 中断 | DMA |
|------|------|-----|
| 角色 | 系统核心模块 | 系统核心模块 |
| 本质 | 把 CPU 部分功能下沉 | 把 CPU 部分功能下沉 |
| 类比 | 大宰相（NVIC） | 小秘书（DMA 控制器） |
| 与外设关系 | 各外设都可产生中断请求 | 各外设都可发出 DMA 请求 |

### 1.2 核心区别

| 对比项 | 中断 | DMA |
|--------|------|-----|
| **功能** | 打断主程序，处理突发事件 | 外设与存储器之间高速数据搬运 |
| **CPU 参与** | 必须参与（执行中断服务程序） | **不参与**（硬件自动完成） |
| **处理内容** | 任意逻辑（软件实现） | 数据传输（硬件实现） |
| **响应方式** | 软件响应（查向量表、执行代码） | 硬件响应（建立通道、自动传输） |

### 1.3 系统架构中的位置

```
              ┌──────────────────────────────────┐
              │          AHB 系统高速总线          │
              │                                  │
  ┌───────┐   │  ┌────────┐   ┌───────────────┐  │
  │  CPU  │───┤  │  NVIC  │   │ DMA 控制器     │  │
  │ (CM3) │   │  │(大宰相) │   │ (小秘书)       │  │
  └───────┘   │  └────────┘   └───────┬───────┘  │
              │                       │          │
              │  ┌────────────────────┼────────┐ │
              │  │  总线矩阵           │        │ │
              └──┴────────────────────┴────────┴─┘
                   │         │         │
                ┌──┴──┐  ┌──┴──┐  ┌───┴───┐
                │ SRAM│  │Flash│  │ 外设   │
                │(RAM)│  │(ROM)│  │(串口等)│
                └─────┘  └─────┘  └───────┘
```

## 四、外设地址自增规则

### 4.1 通用原则

```
外设地址：一般不自增
  → 外设数据寄存器（如 DR）地址固定
  → 自增会偏移到其他寄存器 → 写错位置

存储器地址：一般自增
  → 数组在内存中连续存放
  → 自增可以遍历数组元素
```

### 4.2 各场景配置

| 场景 | PINC（外设自增） | MINC（存储器自增） |
|------|:----------------:|:------------------:|
| 串口发送（RAM→USART） | **0**（DR 地址固定） | **1**（数组遍历） |
| 串口接收（USART→RAM） | **0**（DR 地址固定） | **1**（数组遍历） |
| ROM→RAM（MEM2MEM） | **1**（ROM 数组遍历） | **1**（RAM 数组遍历） |

## 五、作业：RAM 作为外设，ROM 作为存储器

### 5.1 问题描述

```
默认配置（ROM 当外设）：
  ROM → RAM
  CPAR = ROM 地址（外设=源）
  CMAR = RAM 地址（存储器=目的）
  DIR = 0（从外设读）

作业：交换角色（RAM 当外设）
  ROM → RAM（传输方向不变！）
  CPAR = RAM 地址（外设=目的）
  CMAR = ROM 地址（存储器=源）
  DIR = 1（从存储器读，写往外设）
```

### 5.2 修改对照

| 配置项 | 默认（ROM=外设） | 交换后（RAM=外设） |
|--------|:----------------:|:------------------:|
| **DIR** | **0**（从外设读） | **1**（从存储器读） |
| **CPAR** | **ROM 地址**（源） | **RAM 地址**（目的） |
| **CMAR** | **RAM 地址**（目的） | **ROM 地址**（源） |
| 函数调用 | `Transmit(RomAddr, RamAddr, N)` | `Transmit(RomAddr, RamAddr, N)` **不变** |

### 5.3 关键理解

```
函数调用时参数不变：
  源地址（src_addr）= ROM 地址
  目的地址（dest_addr）= RAM 地址

底层赋值交换：
  // 默认（ROM=外设）：
  CPAR = src_addr;      // ROM → 外设
  CMAR = dest_addr;     // RAM → 存储器
  DIR  = 0;             // 从外设读（从ROM读）

  // 交换后（RAM=外设）：
  CPAR = dest_addr;     // RAM → 外设
  CMAR = src_addr;      // ROM → 存储器
  DIR  = 1;             // 从存储器读（从ROM读）

两种方式结果完全相同：都是 ROM → RAM 的传输
只是底层"谁是外设、谁是存储器"的定义不同
```

### 5.4 代码修改

```c
// 默认方式（ROM=外设）
DMA1_Channel1->CPAR  = src_addr;       // ROM 地址给外设
DMA1_Channel1->CMAR  = dest_addr;      // RAM 地址给存储器
DMA1_Channel1->CCR  &= ~DMA_CCR1_DIR; // DIR = 0（从外设读）

// 交换方式（RAM=外设）
DMA1_Channel1->CPAR  = dest_addr;      // RAM 地址给外设
DMA1_Channel1->CMAR  = src_addr;       // ROM 地址给存储器
DMA1_Channel1->CCR  |= DMA_CCR1_DIR;  // DIR = 1（从存储器读）
```

### 5.5 为什么必须同时改三处

```
只改 DIR 不改地址：
  DIR=1（从存储器读→写往外设）
  CPAR=ROM地址（外设=目的）
  CMAR=RAM地址（存储器=源）
  
  → 从 RAM 读 → 写入 ROM ❌（方向反了）

只改地址不改 DIR：
  DIR=0（从外设读→写入存储器）
  CPAR=RAM地址（外设=源）
  CMAR=ROM地址（存储器=目的）
  
  → 从 RAM 读 → 写入 ROM ❌（方向反了）

必须同时改 DIR 和地址：
  DIR=1（从存储器读→写往外设）
  CPAR=RAM地址（外设=目的）
  CMAR=ROM地址（存储器=源）
  
  → 从 ROM 读 → 写入 RAM ✅
```

## 六、MEM2MEM 模式的本质

### 6.1 DMA 底层只有两个角色

```
DMA 控制器底层：
  → 只认识"外设"和"存储器"两个角色
  → 所有寄存器名字都叫：P 开头 = 外设，M 开头 = 存储器
  → 存储器到存储器模式只是把其中一个存储器"伪装"成外设

谁当外设、谁当存储器：
  → 由用户决定
  → 配合 DIR 位调整传输方向
  → 最终效果相同
```

### 6.2 两种配置方式对比

| 配置方式 | 外设角色 | 存储器角色 | DIR | 效果 |
|----------|---------|-----------|:---:|------|
| 方式 A | ROM | RAM | 0 | ROM → RAM |
| 方式 B | RAM | ROM | 1 | ROM → RAM |

> 两种方式完全等价，只是底层角色定义不同。

## 七、概念速查

| 概念       | 说明                           |
| -------- | ---------------------------- |
| DMA      | Direct Memory Access，直接存储器访问 |
| DMA 控制器  | 管理 DMA 通道的硬件（小秘书）            |
| NVIC     | 中断管理器（大宰相）                   |
| MEM2MEM  | 存储器到存储器传输模式                  |
| CCR      | 通道配置寄存器（核心）                  |
| CPAR     | 通道外设地址寄存器                    |
| CMAR     | 通道存储器地址寄存器                   |
| CNDTR    | 通道数据数量寄存器                    |
| DIR      | 数据传输方向（0=外设→存储器）             |
| PINC     | 外设地址自增                       |
| MINC     | 存储器地址自增                      |
| TC       | Transfer Complete，传输完成       |
| ISR      | 中断状态寄存器（只读）                  |
| IFCR     | 中断标志清除寄存器（只写 1 清除）           |
| CR3.DMAT | 串口 DMA 发送使能                  |
| CIRC     | 循环模式                         |
