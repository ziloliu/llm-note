# 高级定时器有限次数 PWM 输出 — 寄存器方式实现 笔记

---

## 一、实验概述

| 项目 | 说明 |
|------|------|
| 定时器 | **TIM1**（高级定时器） |
| 功能 | 输出 5 个周期的 PWM 方波后停止 |
| 输出引脚 | **PA8**（TIM1_CH1） |
| PWM 频率 | **2Hz**（周期 0.5s） |
| 占空比 | **50%** |
| 闪烁次数 | **5 次** |
| 总时长 | 2.5 秒 |

---

## 二、关键参数计算

```
内部时钟 = 72MHz

PSC = 7199 → 分频后频率 = 72MHz / 7200 = 10kHz
  → 每次计数 = 0.1ms = 100μs

ARR = 4999 → 5000 次计数溢出
  → 一个 PWM 周期 = 5000 × 0.1ms = 500ms = 0.5s
  → PWM 频率 = 1 / 0.5 = 2Hz

CCR1 = 2500 → 占空比 = 2500 / 5000 = 50%
  → 高电平 250ms，低电平 250ms

RCR = 4 → 重复计数 4 次
  → 更新事件在第 4+1=5 次溢出时产生
  → 闪烁 5 次后停止
```

---

## 三、GPIO 配置（PA8）

```
PA8 → TIM1_CH1 复用推挽输出
  → CNF = 10（复用推挽）
  → MODE = 11（输出模式，50MHz）
  → 配置寄存器：CRH（高位寄存器）
```

### 3.1 寄存器配置

```c
// 开启时钟
RCC->APB2ENR |= RCC_APB2ENR_IOPAEN;   // GPIOA 时钟
RCC->APB2ENR |= RCC_APB2ENR_TIM1EN;    // TIM1 时钟（⚠️ APB2）

// PA8 配置：复用推挽输出
GPIOA->CRH &= ~(0xF << 0);             // 清零 PA8 对应位 [3:0]
GPIOA->CRH |=  (0xB << 0);             // CNF=10, MODE=11
```

### 3.2 APB 总线差异

| 定时器 | 连接总线 | 时钟使能寄存器 |
|--------|:--------:|---------------|
| TIM1/8 | **APB2**（高速） | **RCC->APB2ENR** |
| TIM2~7 | APB1（低速） | RCC->APB1ENR |

## 四、时基配置

### 4.1 PSC、ARR、RCR

```c
// 预分频器
TIM1->PSC = 7199;           // 7200 分频 → 10kHz

// 自动重装载值
TIM1->ARR = 4999;           // 5000 次计数溢出 → 0.5s

// 计数方向
TIM1->CR1 &= ~TIM_CR1_DIR; // 向上计数（默认）

// 重复计数器（高级定时器特有）
TIM1->RCR = 4;              // 重复 4 次 → 第 5 次溢出产生更新事件
```

### 4.2 RCR 工作原理

```
RCR = 4 时的计数过程：

溢出次数:  第1次   第2次   第3次   第4次   第5次
           ↓      ↓      ↓      ↓      ↓
RCR 递减:   4→3    3→2    2→1    1→0    0→产生更新事件
PWM 输出:   ✅      ✅      ✅      ✅      ✅
                                                  ↓
                                              进入中断 → 停止

闪烁次数 = RCR + 1 = 5
```

## 五、输出通道配置（通道 1）

### 5.1 配置流程

```
① 配置通道 1 为输出模式 → CCMR1.CC1S = 00
② 配置输出模式为 PWM 模式 1 → CCMR1.OC1M = 110
③ 配置 CCR1 占空比 → TIM1->CCR1 = 2500
④ 配置极性（高电平有效）→ CCER.CC1P = 0
⑤ 使能通道 1 输出 → CCER.CC1E = 1
⑥ 使能主输出 → BDTR.MOE = 1
```

### 5.2 寄存器配置

```c
// ① 通道 1 输出模式
TIM1->CCMR1 &= ~TIM_CCMR1_CC1S;      // CC1S = 00（输出）

// ② PWM 模式 1
TIM1->CCMR1 &= ~TIM_CCMR1_OC1M;      // 清零 OC1M
TIM1->CCMR1 |=  (0x6 << 4);           // OC1M = 110（PWM 模式 1）

// ③ 占空比 50%
TIM1->CCR1 = 2500;                     // 2500/5000 = 50%

// ④ 高电平有效
TIM1->CCER &= ~TIM_CCER_CC1P;         // CC1P = 0

// ⑤ 通道 1 使能
TIM1->CCER |=  TIM_CCER_CC1E;         // CC1E = 1

// ⑥ 主输出使能（高级定时器必须）
TIM1->BDTR |=  TIM_BDTR_MOE;          // MOE = 1
```

### 5.3 MOE（主输出使能）

```
⚠️ 高级定时器特有，通用/基本定时器无此位

BDTR.MOE = 0：所有输出强制无效（安全状态）
BDTR.MOE = 1：正常输出

必须置 1，否则无任何输出
位置：BDTR（刹车和死区寄存器）的第 15 位
```

## 六、中断配置

### 6.1 更新中断使能

```c
// 开启更新中断
TIM1->DIER |= TIM_DIER_UIE;           // UIE = 1
```

### 6.2 NVIC 配置

```c
// 优先级分组
NVIC_SetPriorityGrouping(3);           // 全部抢占优先级

// 设置优先级
NVIC_SetPriority(TIM1_UP_IRQn, 2);    // ⚠️ TIM1_UP_IRQn（不是 TIM1_IRQn）

// 使能中断
NVIC_EnableIRQ(TIM1_UP_IRQn);
```

### 6.3 TIM1 专用中断号

```
高级定时器 TIM1 有多个专用中断号：

中断号                    功能
TIM1_UP_IRQn             更新中断（溢出/RCR）     ← 本实验使用
TIM1_BRK_IRQn            刹车中断
TIM1_TRG_COM_IRQn        触发/换相中断
TIM1_CC_IRQn             捕获/比较中断

通用定时器只有一个：TIMx_IRQn
高级定时器按功能拆分，更精细
```

## 七、更新事件与中断的时序问题

### 7.1 问题现象

```
现象：LED 不闪烁，串口显示一启动就进入中断

原因：
  → PSC 和 RCR 有影子寄存器，无控制位
  → 初始化时写入的是预装载值，影子值未刷新
  → PSC 影子值 = 0（72MHz 直接计数）
  → RCR 影子值 = 0（不重复，溢出即更新）
  → 极短时间内完成 5000 次计数 + 更新事件
  → 立刻进入中断 → 停止定时器
```

### 7.2 影子寄存器状态

| 寄存器 | 有无控制位 | 初始影子值 | 预装载值 |
|--------|:----------:|:----------:|:--------:|
| ARR | ✅ ARPE | 默认=预装载值（ARPE=0 直通） | 4999 |
| CCR1 | ✅ OC1PE | 默认=预装载值（OC1PE=1 时缓冲） | 2500 |
| **PSC** | **❌ 无** | **0**（未刷新） | 7199 |
| **RCR** | **❌ 无** | **0**（未刷新） | 4 |

### 7.3 问题时序

```
启动计数器
    │
    ├─ PSC 影子值 = 0 → 计数频率 = 72MHz
    ├─ ARR 影子值 = 4999 → 5000 次计数溢出
    ├─ RCR 影子值 = 0 → 溢出即产生更新事件
    │
    └─ 5000 / 72MHz ≈ 0.069ms → 立刻更新事件 → 进中断 → 停止

    总时间不到 0.1ms，人眼完全看不到
```

## 八、解决方案：手动产生更新事件

### 8.1 方案一：UG + 清除 UIF

```c
// 在初始化末尾、使能中断之后、开启定时器之前

// ① 手动产生更新事件 → 刷新所有影子寄存器
TIM1->EGR |= TIM_EGR_UG;              // UG = 1

// ② 清除更新标志 → 防止进入中断
TIM1->SR &= ~TIM_SR_UIF;              // UIF = 0
```

### 8.2 方案一执行流程

```
初始化配置（写入预装载值）
    │
    ├─ EGR.UG = 1 → 产生更新事件
    │   → PSC 影子值 = 7199 ✅
    │   → RCR 影子值 = 4 ✅
    │   → SR.UIF = 1（更新标志置位）
    │
    ├─ SR.UIF = 0 → 清除标志
    │
    ├─ 开启定时器 CR1.CEN = 1
    │
    └─ 正常计数，5 次后进中断 → 停止
```

### 8.3 方案二：CR1.URS 位控制

```c
// 在产生 UG 之前，设置 URS 位
TIM1->CR1 |= TIM_CR1_URS;             // URS = 1

// 产生更新事件
TIM1->EGR |= TIM_EGR_UG;              // UG = 1
// 此时 UG 不会产生中断（因为 URS=1）
// 影子寄存器已刷新

// 恢复 URS（可选，也可以不恢复）
// TIM1->CR1 &= ~TIM_CR1_URS;
```

### 8.4 CR1.URS 详解

| URS 值 | 行为 |
|:------:|------|
| **0**（默认） | 计数器溢出 **和** UG 都会产生更新中断 |
| **1** | **只有**计数器溢出产生更新中断，UG **不**产生中断 |

```
URS = 1 时：
  → EGR.UG = 1 → 只刷新影子寄存器，不进中断
  → 计数器溢出（RCR 减到 0 后再溢出）→ 正常进中断

完美解决"一启动就进中断"的问题
```

### 8.5 两种方案对比

| 对比项 | 方案一（UG+清UIF） | 方案二（URS） |
|--------|:------------------:|:-------------:|
| 操作 | UG=1，然后 UIF=0 | URS=1，然后 UG=1 |
| 时序风险 | 有微小窗口（UG后UIF前可能进中断） | 无 |
| 代码简洁度 | 两行 | 一行 |
| 推荐程度 | 一般 | **推荐** |

## 九、中断服务函数

```c
void TIM1_UP_IRQHandler(void)
{
    // 清除中断标志
    TIM1->SR &= ~TIM_SR_UIF;

    // 停止定时器
    TIM1->CR1 &= ~TIM_CR1_CEN;        // CEN = 0
}
```

## 十二、完整初始化代码

```c
void TIM1_Init(void)
{
    // ① 开启时钟
    RCC->APB2ENR |= RCC_APB2ENR_IOPAEN;    // GPIOA
    RCC->APB2ENR |= RCC_APB2ENR_TIM1EN;     // TIM1（⚠️ APB2）

    // ② PA8 复用推挽输出
    GPIOA->CRH &= ~(0xF << 0);
    GPIOA->CRH |=  (0xB << 0);              // CNF=10, MODE=11

    // ③ 时基配置
    TIM1->PSC  = 7199;                       // 10kHz
    TIM1->ARR  = 4999;                       // 0.5s 周期
    TIM1->CR1 &= ~TIM_CR1_DIR;              // 向上计数
    TIM1->RCR  = 4;                          // 重复 4 次（5 次闪烁）

    // ④ 输出通道配置
    TIM1->CCMR1 &= ~TIM_CCMR1_CC1S;         // 输出模式
    TIM1->CCMR1 &= ~TIM_CCMR1_OC1M;
    TIM1->CCMR1 |=  (0x6 << 4);             // PWM 模式 1
    TIM1->CCR1   = 2500;                     // 50% 占空比
    TIM1->CCER  &= ~TIM_CCER_CC1P;          // 高电平有效
    TIM1->CCER  |=  TIM_CCER_CC1E;          // 通道 1 使能
    TIM1->BDTR  |=  TIM_BDTR_MOE;           // 主输出使能

    // ⑤ 中断配置
    TIM1->DIER |= TIM_DIER_UIE;             // 更新中断使能
    NVIC_SetPriorityGrouping(3);
    NVIC_SetPriority(TIM1_UP_IRQn, 2);
    NVIC_EnableIRQ(TIM1_UP_IRQn);

    // ⑥ 刷新影子寄存器（方案二）
    TIM1->CR1 |= TIM_CR1_URS;               // URS = 1
    TIM1->EGR |= TIM_EGR_UG;                // 手动更新（不进中断）
}
```

## 十三、CR2.IDLE STATE 配置

### 13.1 问题

```
停止定时器后 LED 保持亮（低电平）
原因：定时器停止后输出引脚保持最后状态
```

### 13.2 解决

```c
// 空闲状态设为高电平（LED 熄灭）
TIM1->CR2 |= TIM_CR2_OIS1;    // OIS1 = 1（空闲高电平）

// 极性改为低电平有效
TIM1->CCER |= TIM_CCER_CC1P;  // CC1P = 1（低电平有效）
```

| 配置 | 修改前 | 修改后 | 效果 |
|------|:------:|:------:|------|
| OC Idle State | Reset（低） | **Set（高）** | 停止后输出高电平 |
| CH Polarity | High（高有效） | **Low（低有效）** | 低电平点亮 LED |

## 十四、与之前案例的对比

### 14.1 双灯闪烁同步问题

```
之前案例：EX07 双灯闪烁
  → 两个 LED 不同步（一个先亮，另一个先灭）
  → 原因：启动时 PSC 影子值未刷新 → 立刻进中断
  → 解决：URS + UG（或 UIF 清除）

本实验：TIM1 有限次数 PWM
  → 同样的问题
  → 同样的解决方案
```

### 14.2 URS/UG 适用场景

```
URS + UG 方案适用于所有需要：
  ① 启动前刷新影子寄存器
  ② 但不希望启动时就进中断

的场景，包括：
  → 有限次数 PWM 输出
  → 多定时器同步启动
  → 任何使用更新中断的场景
```

## 十六、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| LED 不亮 | MOE 未开启 | BDTR.MOE = 1 |
| 一启动就进中断 | PSC/RCR 影子值未刷新 | URS+UG 或 UIF 清除 |
| 闪烁后 LED 常亮 | 空闲状态为低电平 | CR2.OIS1=1 + CC1P=1 |
| 闪烁次数不对 | RCR 值配错 | RCR = 闪烁次数 - 1 |
| 时钟使能错误 | TIM1 在 APB2 | 改用 APB2ENR |
| 中断号错误 | 用了 TIM1_IRQn | 改用 TIM1_UP_IRQn |
| 占空比错误 | CCR 值算错 | CCR = ARR/2（50%） |

## 十八、PSC/RCR 影子寄存器刷新方案总结

| 方案             | 方法                  |   中断    | 推荐  |
| -------------- | ------------------- | :-----: | :-: |
| 等待自然溢出         | 不做处理                | 立即进中断 ❌ |  ❌  |
| 全局变量跳过         | 第一次进中断不做处理          |  进但不处理  | ⚠️  |
| **UG + 清 UIF** | EGR.UG=1, SR.UIF=0  |  不进 ✅   |  ✅  |
| **URS + UG**   | CR1.URS=1, EGR.UG=1 |  不进 ✅   | ✅✅  |
