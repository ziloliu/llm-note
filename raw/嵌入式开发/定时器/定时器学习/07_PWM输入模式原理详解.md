# PWM 输入模式 — 同时测量周期和占空比 笔记

---

## 一、PWM 输入模式概述

| 项目 | 说明 |
|------|------|
| 本质 | 输入捕获模式的一个**特例** |
| 功能 | 同时测量 PWM 方波的**周期**和**占空比** |
| 通道 | 使用**两个通道**（CH1 + CH2） |
| 信号源 | 同一个引脚输入，分配到两个通道 |
| 关键 | 从模式**复位模式** + **交叉连接** |

---

## 二、核心思路

### 2.1 问题分析

```
占空比 = 高电平时间 / 周期

需要测量两个量：
  ① 周期 T = 连续两个上升沿之间的时间
  ② 高电平时间 t = 一个上升沿到紧接着的下降沿之间的时间

一个通道只能配一种极性（上升沿或下降沿）
→ 需要两个通道分别检测不同边沿
```

### 2.2 解决方案

```
同一信号 → 分成两路：
  通道 1：上升沿检测 → 测周期 T → CCR1
  通道 2：下降沿检测 → 测高电平时间 t → CCR2

占空比 = CCR2 / CCR1
```

---

## 三、信号通路详解

### 3.1 完整信号路径

```
CH1 引脚 (PB6) 输入 PWM 方波
    │
    ↓
输入滤波器（不滤波，ICxF=0）
    │
    ↓
    ├────→ 上升沿检测 (CC1P=0) ──→ TI1FP1
    │                                  │
    │                    ┌─────────────┤
    │                    ↓             ↓
    │              直连(CC1S=01)   触发输入(TS=101)
    │                    ↓             ↓
    │               IC1 → CCR1    TRGI → 从模式控制器
    │               (测周期 T)     (SMS=100,复位模式)
    │                                   │
    │                                   ↓
    │                              CNT 清零（硬件自动）
    │
    └────→ 下降沿检测 (CC2P=1) ──→ TI2FP2
                                        │
                                  交叉(CC2S=10)
                                        ↓
                                   IC2 → CCR2
                                   (测高电平时间 t)
```

### 3.2 通道配置对比

| 配置项 | 通道 1 | 通道 2 |
|--------|:------:|:------:|
| 信号来源 | CH1 引脚（直连） | CH1 引脚（交叉） |
| CCMR.CCxS | **01**（直通） | **10**（交叉） |
| CCER.CCxP | **0**（上升沿） | **1**（下降沿） |
| 捕获目标 | CCR1 = **周期 T** | CCR2 = **高电平时间 t** |

## 四、时序分析

### 4.1 时序图

```
PWM 输入信号：
  ┌───┐           ┌───┐           ┌───┐
  │   │           │   │           │   │
──┘   └───────────┘   └───────────┘   └───
  ↑       ↑       ↑       ↑       ↑
  ①      ②       ③      ④       ⑤

CNT 计数器：
  ① CNT=0（复位清零）     ──→ 开始计数
  ② CNT=2（下降沿）       ──→ CCR2=2（高电平时间）
  ③ CNT=4（上升沿）       ──→ CCR1=4（周期）+ CNT=0（复位）
  ④ CNT=2（下降沿）       ──→ CCR2=2（高电平时间）
  ⑤ CNT=4（上升沿）       ──→ CCR1=4（周期）+ CNT=0（复位）
```

### 4.2 详细步骤

| 步骤 | 事件 | 硬件动作 | 结果 |
|:----:|------|----------|------|
| ① | 上升沿到来 | CCR1 捕获 CNT + CNT 复位清零 | CCR1=上次周期, CNT=0 |
| — | 计数 | CNT 递增 0→1→2 | — |
| ② | 下降沿到来 | CCR2 捕获 CNT | CCR2=2（高电平时间） |
| — | 计数 | CNT 递增 3→4 | — |
| ③ | 上升沿到来 | CCR1 捕获 CNT + CNT 复位清零 | CCR1=4（周期）, CNT=0 |

### 4.3 为什么不会捕获到第二次下降沿

```
关键：上升沿复位在下降沿之前到来

时间线：
  上升沿① → 下降沿② → 上升沿③ → 下降沿④ → 上升沿⑤
     │          │          │
     └─ CCR2=2  └─ CCR1=4  └─ CCR2=2
     └─ CNT=0   └─ CNT=0   └─ CNT=0

③ 的上升沿在 ④ 的下降沿之前到来
→ ③ 到来时 CNT 被复位为 0
→ CNT 从 0 重新开始计数
→ 等到 ④ 的下降沿到来时，CCR2 再次捕获正确的高电平时间
→ 永远只测量"从复位开始到第一个下降沿"的时间
```

## 五、各寄存器配置汇总

### 5.1 CCMR1（捕获/比较模式寄存器 1）

| 位域 | 通道 1 配置 | 通道 2 配置 | 说明 |
|------|:-----------:|:-----------:|------|
| CC1S[1:0] | **01** | — | 通道 1 直通 |
| IC1F[3:0] | 0000 | — | 不滤波 |
| IC1PSC[1:0] | 00 | — | 不分频 |
| CC2S[1:0] | — | **10** | 通道 2 交叉 |
| IC2F[3:0] | — | 0000 | 不滤波 |
| IC2PSC[1:0] | — | 00 | 不分频 |

### 5.2 CCER（捕获/比较使能寄存器）

| 位 | 通道 1 | 通道 2 | 说明 |
|:--:|:------:|:------:|------|
| CCxP | **0** | **1** | 上升沿 / 下降沿 |
| CCxE | **1** | **1** | 两个通道都使能 |

### 5.3 SMCR（从模式控制寄存器）

| 位域 | 值 | 说明 |
|------|:--:|------|
| TS[2:0] | **101** | 触发源选择 TI1FP1 |
| SMS[2:0] | **100** | 从模式选择复位模式 |

### 5.4 配置流程（按寄存器）

```
① SMCR.TS  = 101    → 选择 TI1FP1 作为触发输入
② SMCR.SMS = 100    → 选择复位模式

③ CCMR1.CC1S  = 01  → 通道 1 直通模式
④ CCMR1.CC2S  = 10  → 通道 2 交叉模式
⑤ CCER.CC1P   = 0   → 通道 1 上升沿
⑥ CCER.CC2P   = 1   → 通道 2 下降沿
⑦ CCER.CC1E   = 1   → 通道 1 使能
⑧ CCER.CC2E   = 1   → 通道 2 使能

⑨ PSC = 71         → 1MHz 计数频率
⑩ ARR = 65535       → 最大计数范围
```

## 六、复位模式的关键优势

### 6.1 硬件自动清零

```
传统方式（中断清零）：
  上升沿 → 输入捕获事件 → 中断 → 软件执行 CNT=0
  延迟：若干时钟周期（软件响应）

PWM 输入模式（硬件清零）：
  上升沿 → 输入捕获事件（CCR1 快照）+ 复位模式（CNT=0）
  延迟：1 个时钟周期（硬件瞬间完成）

优势：
  ✅ 无需中断
  ✅ 无软件延迟
  ✅ 精度更高
  ✅ 不占用 CPU 资源
```

### 6.2 事件处理顺序保证

```
同一个上升沿同时触发两个硬件动作：

① 输入捕获事件：CCR1 = CNT（快照）  ← 先执行
② 复位模式：     CNT = 0（清零）      ← 后执行

硬件保证：捕获先于复位
→ CCR1 中保存的是清零前的正确周期值
→ 不会出现 CCR1=0 的错误
```

### 6.3 自动周期性更新

```
无需任何软件干预：
  → CCR1 自动保存最新周期值
  → CCR2 自动保存最新高电平时间值
  → 每个 PWM 周期自动更新一次
  → 主循环只需读取 CCR1 和 CCR2 即可
```

## 八、关键概念速查

| 概念 | 说明 |
|------|------|
| **PWM 输入模式** | 输入捕获的特例，用两个通道 + 从模式复位 |
| **直连（Direct）** | CCxS=01，本通道信号给本通道 CCR |
| **交叉（Indirect）** | CCxS=10，其他通道信号给本通道 CCR |
| **TI1FP1** | 通道 1 经滤波+极性选择后的信号 |
| **TI2FP2** | 通道 2 经滤波+极性选择后的信号 |
| **从模式复位** | TRGI 上升沿 → CNT 硬件自动清零 |
| **CCR1** | 保存周期值（两次上升沿之间） |
| **CCR2** | 保存高电平时间值（上升沿到下降沿） |
| **占空比** | CCR2 / CCR1 × 100% |

---

## 九、计算公式

```
CCR1 = 周期 T 的计数值（μs）
CCR2 = 高电平时间 t 的计数值（μs）

周期 = CCR1 / 1000.0           （ms）
频率 = 1000000.0 / CCR1         （Hz）
占空比 = CCR2 / CCR1 × 100%    （%）

例：
  CCR1 = 10000（10ms 周期）
  CCR2 = 6000（6ms 高电平）
  → 周期 = 10.00ms
  → 频率 = 100.00Hz
  → 占空比 = 60.00%
```

## 十、时序图（带 CCR 更新）

```
PWM 信号：  ┌──────┐                ┌──────┐
            │      │                │      │
       ─────┘      └────────────────┘      └──────
            ↑      ↑                ↑      ↑
            ①      ②                ③      ④

CNT：       0→→→→→→2               0→→→→→→2
                   ↗继续              ↗继续
              3→→→→4                3→→→→4

CCR1：           (上次)                  4(周期)
CCR2：           2(高电平)               2(高电平)

动作：
  ① 上升沿：CCR1=上次周期 + CNT=0（复位）
  ② 下降沿：CCR2=2（高电平时间）
  ③ 上升沿：CCR1=4（周期）+ CNT=0（复位）
  ④ 下降沿：CCR2=2（高电平时间）

主循环读取：
  周期 = CCR1 = 4
  高电平 = CCR2 = 2
  占空比 = 2/4 = 50%
```
