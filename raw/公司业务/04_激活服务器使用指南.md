## 一、安装 WSL

### 1.1 启用 Windows 功能

以**管理员身份**打开 PowerShell，手动启用所需组件：

```powershell
# 启用适用于 Linux 的 Windows 子系统
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

# 启用虚拟机平台
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

# 重启电脑
Restart-Computer
```

### 1.2 一键安装 WSL

> 适用于 Windows 10 (2004+) / Windows 11

```powershell
wsl --install --no-distribution
```

该命令会自动完成以下操作：

- 启用「适用于 Linux 的 Windows 子系统」功能
- 启用「虚拟机平台」功能
- 下载并安装 WSL 2 Linux 内核
- 将 WSL 2 设置为默认版本

`--no-distribution` 参数表示**不会自动安装任何发行版**（如 Ubuntu），后续可按需单独安装。

### 1.3 验证安装

```powershell
# 查看 WSL 版本信息
wsl --version

# 确认当前无已安装的发行版
wsl --list -v
```

预期输出：

```
WSL 版本: 2.x.x.x
内核版本: ...
```

```
适用于 Linux 的 Windows 子系统没有已安装的分发。
```

### 1.4 常见问题：提示需要重启

```powershell
# 如果执行后提示 "请求的操作需要提升/重新启动"
Restart-Computer

# 重启后可能需要再执行一次
wsl --install --no-distribution
```

## 二、WSL 环境迁移

将一台电脑上的 WSL 发行版完整迁移到另一台电脑，包括已安装的软件、配置文件和用户数据。

### 2.1 在源电脑上导出

在 PowerShell 中执行：

```powershell
# 1. 查看已安装的发行版名称
wsl --list -v

# 2. 关闭 WSL，确保数据一致性
wsl --shutdown

# 3. 导出为 tar 文件（将 Ubuntu 替换为你的发行版名称）
wsl --export Ubuntu D:\backup\ubuntu-wsl.tar
```

> 导出的 `.tar` 文件是整个文件系统的完整快照。

### 2.2 传输文件

将 `ubuntu-wsl.tar` 拷贝到目标电脑。
* 建议U盘，镜像文件较大（50G以上）
* 现有评估环境的WSL镜像保存在
	* WIN1:"E:\software\ubuntu-wsl.tar"
	* WIN2:"C:\projects\softwares\ubuntu-wsl.tar"

### 2.3 在目标电脑上导入

```powershell
# 1. 导入发行版（指定安装路径，避免占用 C 盘）
wsl --import Ubuntu D:\WSL\Ubuntu D:\backup\ubuntu-wsl.tar

# 2. 启动并验证
wsl -d Ubuntu
```

> 目标电脑需先完成 [第一节](#一安装-wsl) 的 WSL 安装步骤。

### 2.4 迁移后修复

导入后默认登录用户会变成 `root`，需要手动修复。

**步骤一：编辑 `wsl.conf`**

```bash
sudo nano /etc/wsl.conf
```

写入以下内容（将 `yourusername` 替换为原来的用户名）：

```ini
[user]
default=yourusername
```

保存并退出（`Ctrl+O` → `Enter` → `Ctrl+X`）。

**步骤二：重启 WSL 发行版**

```powershell
wsl --shutdown
wsl -d Ubuntu
```

重启后即恢复为原来的用户身份登录。

## 附录：快速参考

| 操作 | 命令 |
|---|---|
| 查看已安装发行版 | `wsl --list -v` |
| 关闭所有 WSL 实例 | `wsl --shutdown` |
| 导出发行版 | `wsl --export <名称> <路径.tar>` |
| 导入发行版 | `wsl --import <名称> <安装路径> <tar路径>` |
| 启动指定发行版 | `wsl -d <名称>` |
| 查看 WSL 版本 | `wsl --version` |
