## 一、定时器分类总览

STM32F103 系列共有 **11 个定时器**，分为 4 大类：

| 类型 | 数量 | 名称 | 位宽 | 特点 |
|------|:----:|------|:----:|------|
| **基本定时器** | 2 | TIM6, TIM7 | 16位 | 功能最少（低配版） |
| **通用定时器** | 4 | TIM2, TIM3, TIM4, TIM5 | 16位 | 功能较全（标准版） |
| **高级定时器** | 2 | TIM1, TIM8 | 16位 | 功能最全（高配版） |
| **系统滴答定时器** | 1 | SysTick | 24位 | 系统时钟，向下计数 |
| **看门狗定时器** | 2 | IWDG, WWDG | — | 故障检测，系统安全 |

## 二、三类通用定时器对比

```
功能丰富程度：

基本定时器（TIM6/TIM7）  ▰▰▱▱▱▱▱▱▱▱  低配版
通用定时器（TIM2~TIM5）  ▰▰▰▰▰▰▱▱▱▱  标准版
高级定时器（TIM1/TIM8）  ▰▰▰▰▰▰▰▰▰▰  高配版
```

### 2.1 基本定时器（TIM6 / TIM7）

| 特性 | 说明 |
|------|------|
| 位宽 | 16 位 |
| 功能 | 仅定时功能 |
| 主要用途 | 驱动 DAC 等外设 |
| 特点 | 功能最少，最简单 |

### 2.2 通用定时器（TIM2 / TIM3 / TIM4 / TIM5）

| 特性 | 说明 |
|------|------|
| 位宽 | 16 位 |
| 功能 | 定时 + 输入捕获 + 输出比较 + PWM + 脉冲计数 |
| 通道数 | 每个定时器多达 **4 个**通道 |
| 主要用途 | 定时中断、PWM 输出、输入信号测量、脉冲计数 |

### 2.3 高级定时器（TIM1 / TIM8）

| 特性 | 说明 |
|------|------|
| 位宽 | 16 位 |
| 功能 | 通用定时器全部功能 **+ 死区控制 + 紧急刹车** |
| 主要用途 | **电机控制** PWM 波形输出 |
| 特点 | 在通用定时器基础上增加高级控制功能 |

## 四、系统滴答定时器（SysTick）

| 特性 | 说明 |
|------|------|
| 位宽 | **24 位**（与其他定时器不同） |
| 计数方向 | **向下**计数 |
| 归属 | ARM Cortex-M3 内核（非外设） |
| 用途 | 操作系统时钟节拍、延时函数（`Delay_us` / `Delay_ms`） |

> SysTick 与 TIM1~TIM8 是完全不同的定时器，不属于外设定时器。

📄 [原文01: STM32定时器概述](../raw/嵌入式开发/定时器/01_STM32定时器概述.md)

## 五、看门狗定时器（Watchdog）

| 类型 | 全称 | 说明 |
|------|------|------|
| **IWDG** | Independent Watchdog | 独立看门狗 |
| **WWDG** | Window Watchdog | 窗口看门狗 |

| 特性 | 说明 |
|------|------|
| 本质 | 也是一个定时器 |
| 用途 | 故障检测、系统安全保证 |
| 原理 | 定时器超时未"喂狗" → 系统复位 |
| 学习安排 | 放在**高级篇**讲解 |

## 六、定时器功能层层递进关系

```
基本定时器（TIM6/TIM7）
  │
  │  + 输入捕获
  │  + 输出比较
  │  + PWM 输出
  │  + 脉冲计数
  │  + 4 个通道
  ▼
通用定时器（TIM2~TIM5）
  │
  │  + 死区控制
  │  + 紧急刹车
  │  + 互补输出（电机驱动）
  ▼
高级定时器（TIM1/TIM8）
```

> 高级定时器包含通用定时器的全部功能，通用定时器包含基本定时器的全部功能。
> 类似于手机/电脑的低配版、标准版、高配版的区别。
