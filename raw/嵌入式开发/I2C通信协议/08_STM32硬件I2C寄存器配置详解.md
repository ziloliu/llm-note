# STM32 I²C 硬件外设寄存器配置详解 笔记

---

## 一、需要配置的寄存器总览

| 寄存器 | 名称 | 配置内容 | 配置时机 |
|--------|------|----------|----------|
| **CR1** | 控制寄存器 1 | 模式选择(SMBus/I²C)、ACK、START、STOP、PE 使能 | PE 位最后使能 |
| **CR2** | 控制寄存器 2 | 输入时钟频率 (FREQ) | PE 使能前 |
| **CCR** | 时钟控制寄存器 | SCL 频率分频系数、标准/快速模式选择 | PE 使能前 |
| **TRISE** | 上升时间寄存器 | SCL 最大上升沿时间 | PE 使能前 |

---

## 二、CR1 — 控制寄存器 1

### 2.1 关键位

| 位 | 名称 | 功能 | 配置 |
|:--:|------|------|:----:|
| 0 | **PE** | 外设使能（Peripheral Enable） | **最后置 1** |
| 1 | SMBus | SMBus 模式选择 | 0 = I²C 模式 |
| 8 | **START** | 产生起始信号 | 写 1 产生起始 |
| 9 | **STOP** | 产生停止信号 | 写 1 产生停止 |
| 10 | **ACK** | 使能应答 | 1 = 收到字节后自动发 ACK |
| 11 | — | — | — |
| 15 | **SWRST** | 软件复位 | |

### 2.2 关键位详解

**ACK 位：**
```
ACK = 1：每收到一个字节后，硬件自动发送 ACK
ACK = 0：收到字节后不发 ACK（等效于 NACK）
```

**START / STOP 位：**
```
START = 1：硬件自动生成起始信号
STOP  = 1：硬件自动生成停止信号
（与 ACK 不同，START/STOP 写 1 即刻生效）
```

**PE 位（外设使能）：**
```
PE = 0：I²C 外设禁用（可配置寄存器）
PE = 1：I²C 外设使能（CR2、CCR、TRISE 不可再修改）

⚠️ 配置顺序：先配好 CR2、CCR、TRISE，最后再置 PE = 1
```

### 2.3 本实验配置

```c
// 确认 I²C 模式（SMBus = 0，可不配，默认为 0）
I2C2->CR1 &= ~I2C_CR1_SMBTYPE;   // SMBus 位清零 → I²C 模式
// PE 位暂不使能，等所有参数配好后最后开启
```

## 三、CR2 — 控制寄存器 2

### 3.1 FREQ 位（位 5:0）

| 位 | 名称 | 功能 |
|:--:|------|------|
| 5:0 | **FREQ[5:0]** | I²C 输入时钟频率（单位：MHz） |

### 3.2 FREQ 取值范围

| FREQ 值 | 十进制 | 含义 |
|:-------:|:------:|------|
| 000000 | 0 | **禁用** |
| 000001 | 1 | **禁用** |
| 000010 | 2 | 2 MHz（最小可用） |
| ... | ... | ... |
| 100100 | 36 | 36 MHz（APB1 最大频率） |

### 3.3 为什么选 36 MHz

```
I²C1 和 I²C2 均连接在 APB1 低速外设总线上
APB1 最大时钟频率 = 36 MHz
→ FREQ 直接配置为 36（0x24）
```

### 3.4 本实验配置

```c
I2C2->CR2 = 36;   // FREQ = 36，输入时钟 36 MHz
```

> CR2 的高 10 位可配置中断使能、DMA 使能等，本实验暂不使用。

## 四、CCR — 时钟控制寄存器

### 4.1 寄存器位定义

| 位 | 名称 | 功能 |
|:--:|------|------|
| 11:0 | **CCR[11:0]** | 时钟分频系数（12 位） |
| 13:12 | — | 保留 |
| 14 | **DUTY** | 快速模式占空比选择 |
| 15 | **F/S** | 标准/快速模式选择 |

### 4.2 F/S 位

| 值 | 模式 | 数据传输速率 |
|:--:|------|------------|
| **0** | **标准模式** | **100 kbps** |
| 1 | 快速模式 | 400 kbps |

### 4.3 CCR 计算（标准模式）

#### 标准模式公式

```
SCL 高电平时间 = CCR × T_PCLK1
SCL 低电平时间 = CCR × T_PCLK1

（标准模式下高低电平时间相同，占空比 50%）
```

#### 计算过程

```
目标：SCL 频率 = 100 kHz
  → SCL 周期 = 1 / 100kHz = 10 μs
  → 高电平时间 = 10 μs / 2 = 5 μs

输入时钟：APB1 = 36 MHz
  → T_PCLK1 = 1 / 36MHz = 1/36 μs

CCR = 高电平时间 / T_PCLK1
    = 5 μs / (1/36 μs)
    = 5 × 36
    = 180
```

#### 通用公式

```
CCR = (目标高电平时间) × (输入时钟频率 MHz)
    = 5 μs × 36 MHz = 180
```

#### 不同输入时钟的 CCR 值

| 输入时钟 | T_PCLK1 | CCR（标准模式 100kHz） |
|----------|----------|:---------------------:|
| 36 MHz | 1/36 μs | 5 × 36 = **180** |
| 8 MHz | 1/8 μs | 5 × 8 = **40** (0x28) |
| 2 MHz | 1/2 μs | 5 × 2 = **10** |

### 4.4 本实验配置

```c
I2C2->CCR = 180;   // F/S = 0（标准模式），CCR = 180
```

### 4.5 SCL 时钟波形

```
SCL:
    ┌─────┐       ┌─────┐       ┌─────┐
    │     │       │     │       │     │
────┘     └───────┘     └───────┘     └─────
    |←5μs→|←5μs→|
    |←── 10μs ──→|
    |←── 一个 SCL 周期 ──→|

    高电平 5μs = 180 × (1/36)μs
    低电平 5μs = 180 × (1/36)μs
    占空比 = 50%
```

## 五、TRISE — 上升时间寄存器

### 5.1 功能

配置 SCL 信号允许的**最大上升沿时间**，需符合 I²C 总线规范。

### 5.2 数据手册参数

查阅数据手册（电气特性章节）：

| 参数 | 标准模式 I²C | 快速模式 I²C |
|------|:----------:|:-----------:|
| SCL 最大上升时间 (tr) | **1000 ns = 1 μs** | 300 ns |

### 5.3 计算公式

```
TRISE = (最大上升时间 / T_PCLK1) + 1
      = (tr × 输入时钟频率) + 1
```

### 5.4 计算过程（36 MHz）

```
最大上升时间 tr = 1 μs
T_PCLK1 = 1/36 μs

TRISE = 1 μs / (1/36 μs) + 1
      = 36 + 1
      = 37
```

### 5.5 不同输入时钟的 TRISE 值

| 输入时钟 | TRISE（标准模式） |
|----------|:-----------------:|
| 36 MHz | 36 + 1 = **37** |
| 8 MHz | 8 + 1 = **9** |
| 2 MHz | 2 + 1 = **3** |

### 5.6 为什么要 +1

```
手册说明：在计算出的时钟周期数基础上 "increased by one"
含义：多留一个时钟周期的余量，确保上升沿有足够时间完成
```

### 5.7 本实验配置

```c
I2C2->TRISE = 37;   // 36 MHz → 36 + 1 = 37
```

## 六、SCL 时钟时间组成

```
标准模式下 SCL 一个完整周期：

        tr_SCL              tw_SCLH
      (上升时间)         (高电平保持)
         ↗  ┌─────────────┐
SCL: ──────→│             │
         ↑  │             │
         │  │   thigh     │
         │  │             └──────────────┐
         │  │                             │
         └──┘                tw_SCLL      ↘ tf_SCL
                            (低电平保持)    (下降时间)
                             tlow

thigh = tr_SCL + tw_SCLH  （高电平总时间 = 上升沿 + 保持高电平）
tlow  = tf_SCL + tw_SCLL  （低电平总时间 = 下降沿 + 保持低电平）

标准模式：thigh = tlow = CCR × T_PCLK1（50% 占空比）
```

## 七、配置顺序（重要）

```
① 使能时钟
   RCC->APB1ENR |= RCC_APB1ENR_I2C2EN;   // I²C2 时钟
   RCC->APB2ENR |= RCC_APB2ENR_IOPBEN;    // GPIOB 时钟

② 配置 GPIO（PB10/PB11 复用开漏输出）

③ CR2：配置输入时钟频率
   I2C2->CR2 = 36;

④ CCR：配置 SCL 时钟分频（标准模式 100kHz）
   I2C2->CCR = 180;

⑤ TRISE：配置最大上升时间
   I2C2->TRISE = 37;

⑥ CR1：配置模式和 ACK
   I2C2->CR1 &= ~I2C_CR1_SMBTYPE;  // I²C 模式
   I2C2->CR1 |=  I2C_CR1_ACK;      // 使能 ACK

⑦ CR1：最后使能外设
   I2C2->CR1 |=  I2C_CR1_PE;       // 最后开启 PE
```

> **必须先配好所有参数再开启 PE**，PE 使能后 CR2、CCR、TRISE 不可修改。

## 十、寄存器配置公式总结

| 公式 | 说明 |
|------|------|
| `CCR = thigh × f_PCLK1` | 标准模式下，thigh = 5μs |
| `CCR = 5μs × 36MHz = 180` | 本实验 CCR 值 |
| `TRISE = (tr / T_PCLK1) + 1` | tr = 1μs（标准模式） |
| `TRISE = 1μs × 36MHz + 1 = 37` | 本实验 TRISE 值 |
| `FREQ = 输入时钟频率(MHz)` | 直接写数值，单位 MHz |
