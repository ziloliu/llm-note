# 通用定时器输出比较与 PWM 波形生成 笔记

---

## 一、输出比较功能概述

| 项目 | 说明 |
|------|------|
| 功能 | 通过 CNT 与 CCR 比较，控制输出引脚电平 |
| 输出类型 | 数字方波（只有高/低两种电平） |
| 主要应用 | **PWM 波形输出** |
| 通道数 | 每个定时器 **4 个**通道 |
| 引脚复用 | 每个通道的输入和输出共用**同一个引脚** |

---

## 二、输出比较硬件结构

### 2.1 核心组成部分

```
┌──────────────────────────────────────────────┐
│              通用定时器输出比较                  │
│                                              │
│   ① 计数器部分                                │
│      ┌─────┐    ┌─────┐                      │
│      │ PSC │───→│ CNT │                      │
│      └─────┘    └──┬──┘                      │
│                    │                          │
│                    ↓                          │
│   ② 捕获/比较寄存器（每个通道一个）            │
│      ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐ │
│      │CCR1  │  │CCR2  │  │CCR3  │  │CCR4  │ │
│      └──┬───┘  └──┬───┘  └──┬───┘  └──┬───┘ │
│         │         │         │         │      │
│         ↓         ↓         ↓         ↓      │
│   ③ 输出控制电路（每通道一个）                 │
│      ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐ │
│      │ OC1  │  │ OC2  │  │ OC3  │  │ OC4  │ │
│      └──┬───┘  └──┬───┘  └──┬───┘  └──┬───┘ │
│         │         │         │         │      │
│         ↓         ↓         ↓         ↓      │
│        CH1       CH2       CH3       CH4     │
│       (PA6)     (PA7)     (PB0)     (PB1)    │
│                                              │
└──────────────────────────────────────────────┘
```

### 2.2 引脚对应关系（TIM3 为例）

| 通道 | 引脚 | 复用功能 |
|:----:|:----:|----------|
| CH1 | **PA6** | TIM3_CH1 |
| CH2 | **PA7** | TIM3_CH2 |
| CH3 | **PB0** | TIM3_CH3 |
| CH4 | **PB1** | TIM3_CH4 |

> 输入捕获和输出比较共用同一个引脚，配置为**复用推挽输出**即可。

## 三、输出比较工作原理

### 3.1 核心逻辑

```
CNT（计数器）持续递增
       ↓
与 CCR（捕获/比较寄存器）中的基准值比较
       ↓
根据比较结果控制输出引脚电平（高/低）
       ↓
产生方波波形
```

### 3.2 关键寄存器

| 寄存器 | 作用 | 可调性 |
|--------|------|:------:|
| **ARR** | 决定计数上限（PWM 周期/频率） | 固定 |
| **CCR** | 比较基准值（PWM 占空比） | **可调** |
| **CNT** | 当前计数值（自动递增） | 只读 |

## 四、八种输出比较模式

### 4.1 模式总览

| OCM[2:0] | 模式名称 | 比较条件 | 输出行为 |
|:--------:|----------|----------|----------|
| **000** | 冻结 | — | 输出不受比较影响 |
| **001** | 强制输出高电平 | CNT = CCR | 输出高，之后保持 |
| **010** | 强制输出低电平 | CNT = CCR | 输出低，之后保持 |
| **011** | 输出翻转 | CNT = CCR | 每次相等时翻转 |
| **100** | 强制输出低电平 | — | 无条件输出低 |
| **101** | 强制输出高电平 | — | 无条件输出高 |
| **110** | **PWM 模式 1** | CNT < CCR → 高；CNT ≥ CCR → 低 | **最常用** |
| **111** | **PWM 模式 2** | CNT < CCR → 低；CNT ≥ CCR → 高 | 模式 1 取反 |

> OCM 位于 **CCMR1**（捕获/比较模式寄存器）中，3 位控制。

### 4.2 各模式详解

#### 模式 000 — 冻结

```
输出与比较结果完全无关
保持之前的输出状态不变
默认模式
```

#### 模式 001 — 匹配时强制输出高

```
CNT = CCR 时 → 输出高电平，之后保持不变
```

#### 模式 010 — 匹配时强制输出低

```
CNT = CCR 时 → 输出低电平，之后保持不变
```

#### 模式 011 — 匹配时翻转

```
ARR = 99, CCR = 60, 向上计数

输出: ____████████████████████████████████░░░░░░░░░░░░░░░░░░░░████████
CNT:  0        59 60       99→0        59 60       99→0
              ↑             ↑                         ↑
           翻转为高       不翻转                    翻转为低

特点：
  → 周期 = 溢出周期 × 2（频率 = 溢出频率 / 2）
  → 占空比 = 固定 50%
  → CCR 值可调整相位
```

#### 模式 100 — 强制输出低

```
无条件输出低电平，与比较结果无关
```

#### 模式 101 — 强制输出高

```
无条件输出高电平，与比较结果无关
```

#### 模式 110 — PWM 模式 1（最常用）

```
CNT < CCR  → 输出 高电平
CNT ≥ CCR  → 输出 低电平

ARR = 99, CCR = 60, 向上计数

输出: ████████████████████████████████████░░░░░░░░░░░░░░░░░░░░
CNT:  0         59 60           99→0      59 60           99
      |← CCR=60 高电平 →|← ARR+1-CCR=40 低电平 →|

特点：
  → 周期 = 溢出周期（频率 = 溢出频率）
  → 占空比 = CCR / (ARR + 1) = 60 / 100 = 60%
  → CCR 值直接等于占空比百分数
  → 最常用，直觉对应
```

#### 模式 111 — PWM 模式 2

```
CNT < CCR  → 输出 低电平
CNT ≥ CCR  → 输出 高电平

ARR = 99, CCR = 60, 向上计数

输出: ░░░░░░░░░░░░░░░░░░░░████████████████████████████████████
CNT:  0         59 60           99→0      59 60           99

特点：
  → 周期 = 溢出周期
  → 占空比 = 1 - CCR/(ARR+1) = 40%
  → 模式 1 的反相输出
```

## 五、PWM 模式 1 详解（最常用）

### 5.1 波形生成过程

```
ARR = 99（计数 100 次），CCR = 60，PWM 模式 1

CNT:  0  1  2  ...  58  59  60  61  ...  98  99  0  1  2  ...
                   ↑               ↑                ↑
输出: ████████████████████████████░░░░░░░░░░░░░░░░░░░░████████
      |← 60 个高电平 →|← 40 个低电平 →|
      |←────────── 一个完整 PWM 周期 ──────────→|

高电平时间：60 个计数周期
低电平时间：40 个计数周期
PWM 周期：100 个计数周期 = ARR + 1
占空比 = CCR / (ARR + 1) = 60 / 100 = 60%
```

### 5.2 调节占空比

```
固定 ARR = 99（频率不变），改变 CCR 值：

CCR = 0：  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  占空比 = 0%
CCR = 25： ████████████░░░░░░░░░░░░░░░░░░  占空比 = 25%
CCR = 50： ██████████████████░░░░░░░░░░░░  占空比 = 50%
CCR = 75： ████████████████████████░░░░░░  占空比 = 75%
CCR = 100：██████████████████████████████  占空比 = 100%

→ CCR 直接对应占空比百分数，非常直观
```

### 5.3 呼吸灯实现

```
循环改变 CCR 值：

CCR: 0 → 1 → 2 → ... → 99 → 100 → 99 → ... → 1 → 0 → ...
     |←──── 渐亮 ────→|←──── 渐暗 ────→|
     |←────────── 一个呼吸周期 ──────────→|

每次改变 CCR 后，PWM 波形的占空比立即改变
LED 亮度随占空比线性变化
```

## 六、PWM 模式 1 vs 模式 2

| 对比项 | PWM 模式 1 (110) | PWM 模式 2 (111) |
|--------|:----------------:|:----------------:|
| CNT < CCR | **高电平** | 低电平 |
| CNT ≥ CCR | 低电平 | **高电平** |
| 占空比 | **CCR / (ARR+1)** | 1 - CCR/(ARR+1) |
| CCR 含义 | 直接是高电平占比 | 是低电平占比 |
| 使用频率 | **最常用** | 较少 |
| 适用场景 | 一般 PWM 应用 | 需要反相信号时 |

> PWM 模式 1 更直观：CCR 值直接对应高电平占比（占空比），**推荐使用**。

## 七、PWM 关键参数与定时器参数的关系

| PWM 参数 | 定时器参数 | 计算公式 |
|----------|-----------|----------|
| **PWM 频率** | ARR + PSC + 时钟频率 | f = 时钟频率 / ((PSC+1) × (ARR+1)) |
| **PWM 周期** | ARR + PSC + 时钟频率 | T = (PSC+1) × (ARR+1) / 时钟频率 |
| **占空比** | CCR 和 ARR | D = CCR / (ARR + 1) × 100% |
| **分辨率** | ARR | 1 / (ARR + 1) |

### 示例计算

```
目标：PWM 频率 = 1kHz，占空比可调（0~100%），分辨率 = 1%

① PWM 频率 = 1kHz → PWM 周期 = 1ms
② 分辨率 = 1% → ARR + 1 = 100 → ARR = 99
③ 计数频率 = 100 × 1kHz = 100kHz
④ PSC = 72MHz / 100kHz - 1 = 719

验证：
  PWM 频率 = 72MHz / ((719+1) × (99+1)) = 72MHz / 72000 = 1kHz ✅
  占空比 = CCR / 100 × 100%
  CCR = 0  → 0%    → LED 熄灭
  CCR = 50 → 50%   → LED 半亮
  CCR = 100 → 100% → LED 最亮
```

## 八、GPIO 配置

输出比较/PWM 输出时，对应引脚配置为：

| 配置项 | 设置 |
|--------|------|
| 模式 | **复用推挽输出**（Alternate Function Push-Pull） |
| 速度 | High Speed |
| 上下拉 | 无 |

```
为什么是复用推挽输出：
  → 复用：引脚被定时器模块控制，不是通用 GPIO
  → 推挽：需要主动输出高/低电平
  → 输入通道仍然可用（输出模式下输入通道不关闭）
```

## 九、呼吸灯实现思路

```
① 配置定时器为 PWM 模式 1，输出到 LED 引脚
   → PSC = 719（100kHz 计数频率）
   → ARR = 99（1kHz PWM 频率，100 级分辨率）
   → CCR 初始值 = 0

② 启动 PWM 输出

③ 主循环中循环改变 CCR 值：
   while(1)
   {
       for (CCR = 0; CCR <= ARR; CRR++)   // 渐亮
       {
           TIM->CCRx = CCR;
           Delay_ms(N);
       }
       for (CCR = ARR; CCR >= 0; CRR--)   // 渐暗
       {
           TIM->CCRx = CRR;
           Delay_ms(N);
       }
   }

④ LED 亮度随 CCR 变化，实现呼吸效果
```

## 十一、总结

| 项目 | 说明 |
|------|------|
| PWM 模式 1 | CNT < CCR → 高，CNT ≥ CCR → 低 |
| CCR 作用 | 直接决定占空比（百分数） |
| ARR 作用 | 决定 PWM 频率和分辨率 |
| PSC 作用 | 配合 ARR 得到目标 PWM 频率 |
| 呼吸灯 | 固定频率，循环改变 CCR 值 |
| GPIO 配置 | 复用推挽输出 |
