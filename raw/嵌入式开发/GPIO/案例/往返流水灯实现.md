## 一句话定义
在基础流水灯的基础上，实现LED依次点亮后反向依次熄灭的往返流水效果。
## 核心内容
1. 两种实现模式：
   - 模式1：LED1→LED2→LED3→LED2→LED1 循环，符合实际使用需求
   - 模式2：LED1→LED2→LED3→LED3→LED2→LED1 循环，头尾LED点亮时间加倍
2. 通用实现逻辑：
   - 定义LED数组存储所有LED引脚：`uint16_t leds[] = {LED1_Pin, LED2_Pin, LED3_Pin}`
   - 正向遍历点亮LED：`for(uint8_t i=0; i<3; i++)`
   - 反向遍历熄灭LED：掐头去尾从n-2到1，或完整反向遍历
3. 代码示例：
```c
// 依次点亮
for(uint8_t i=0; i<n; i++) {
    LED_On(leds[i]);
    HAL_Delay(500);
}
// 依次熄灭（往返效果）
for(int8_t i=n-1; i>=0; i--) {
    LED_Off(leds[i]);
    HAL_Delay(500);
}
```
## 注意事项&踩坑
1. 反向遍历必须使用有符号整数类型（如int8_t），避免无符号数减到0后溢出变为255导致死循环
2. 若使用无符号数，可通过`leds[n-i-1]`计算反向索引实现相同效果
3. 灯数量较少时（如3个）也可以单独控制中间灯实现折返
