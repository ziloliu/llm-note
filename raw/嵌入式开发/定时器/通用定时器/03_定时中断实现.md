## 三、TIM2 初始化代码

### 3.1 tim2.h

```c
#ifndef __TIM2_H
#define __TIM2_H
#include "stm32f10x.h"
void TIM2_Init(void);
#endif
```

### 3.2 tim2.c

```c
#include "tim2.h"
#include "led.h"

void TIM2_Init(void)
{
    // ① 开启 TIM2 时钟（APB1 低速总线）
    RCC->APB1ENR |= RCC_APB1ENR_TIM2EN;

    // ② 预分频：7200 分频 → 10kHz
    TIM2->PSC = 7199;

    // ③ 自动重装载：计数 10000 次 → 1s
    TIM2->ARR = 9999;

    // ④ 使能更新中断
    TIM2->DIER |= TIM_DIER_UIE;

    // ⑤ 配置 NVIC
    NVIC_SetPriorityGrouping(3);
    NVIC_SetPriority(TIM2_IRQn, 3);
    NVIC_EnableIRQ(TIM2_IRQn);

    // ⑥ 使能定时器（最后）
    TIM2->CR1 |= TIM_CR1_CEN;
}

void TIM2_IRQHandler(void)
{
    if (TIM2->SR & TIM_SR_UIF)
    {
        TIM2->SR &= ~TIM_SR_UIF;   // 清除中断标志
        LED2_Toggle();               // 翻转 LED2
    }
}
```

### 3.3 main.c

```c
#include "stm32f10x.h"
#include "led.h"
#include "tim2.h"

int main(void)
{
    LED_Init();
    TIM2_Init();
    while (1) {}
}
```

## 四、配置流程与寄存器操作

| 步骤 | 寄存器 | 操作 | 值 | 含义 |
|:----:|--------|------|:--:|------|
| ① | RCC->APB1ENR | OR | TIM2EN | 开启 TIM2 时钟 |
| ② | TIM2->PSC | = | 7199 | 7200 分频 → 10kHz |
| ③ | TIM2->ARR | = | 9999 | 计数 10000 次 |
| ④ | TIM2->DIER | OR | UIE | 使能更新中断 |
| ⑤ | NVIC | Enable | TIM2_IRQn | 使能 NVIC 中断 |
| ⑥ | TIM2->CR1 | OR | CEN | 启动计数器 |
| 中断 | TIM2->SR | AND ~UIF | — | 清除中断标志 |

---

## 五、定时参数计算

```
定时时间 = (PSC + 1) × (ARR + 1) / 时钟频率
        = (7199 + 1) × (9999 + 1) / 72MHz
        = 7200 × 10000 / 72000000
        = 72000000 / 72000000
        = 1 秒 ✅
```

## 七、注意事项

| 易错点 | 说明 |
|--------|------|
| APB1 不是 APB2 | TIM2 连接 APB1，不要写 `RCC_APB2ENR` |
| PSC/ARR 不超 65535 | 16 位寄存器限制 |
| 必须配 NVIC | 外设中断需手动配置（与 SysTick 不同） |
| 必须清除 UIF | 软件写 0，否则中断反复触发 |
| 最后开启 CEN | 确保所有参数配置完成后再启动 |

## 八、通用定时器定时中断应用场景

| 应用 | 说明 |
|------|------|
| 周期性任务 | 定时执行特定操作（如采样、控制） |
| 超时检测 | 检测操作是否超时 |
| 精确延时 | 比 SysTick 更灵活的延时 |
| 多任务调度 | 多个定时器实现不同周期的任务 |
| PWM 生成基础 | 定时中断是 PWM 输出的基础 |
