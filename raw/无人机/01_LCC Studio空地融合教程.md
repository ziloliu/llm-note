# LCC Studio 空地融合教程笔记

---

## 一、功能概述

空地融合功能将**无人机航拍数据**与**手持激光扫描仪数据**整合为统一的三维模型，覆盖空中、地面全场景。

---

## 二、设备准备

| 设备    | 要求                |
| ----- | ----------------- |
| 手持扫描仪 | 搭载 RTK 的灵光 k1 扫描仪 |
| 无人机   | 带 RTK 的测绘级无人机     |

---

## 三、采集前勘察

1. 获取测区的**高清卫星影像**或**航拍俯视图**，直观了解环境与重点关注对象。
2. 选取 **4–5 个空旷区域**，均匀分布于测区内，作为**无人机起降点**。
3. 规划整体采集路径。

---

## 四、地面数据采集

1. 启动灵光扫描仪，在 APP 内将坐标系统设置为 **WGS84**。
2. 采集方式与 LCC 常规采集手法相同。
3. **关键要求**：采集路径需**经过每个起降点**，并在每个起降点**半径 2 米范围内完成两次回环采集**。

---

## 五、无人机数据采集

### 核心要求

空地融合中最影响**成功率**和**重建完整度**的因素：

- 测区内所有关注的地面物体，需有**高于 85% 以上重叠度**的倾斜式影像照片。
- 拍照地面物体表面的照片分辨率建议**不低于 1.0 厘米**（分辨率越高，模型越清晰）。

### 飞行设置

| 参数 | 设置值 |
|---|---|
| 坐标系统 | WGS84 |
| 航线模式 | 精灵智能摆拍航线 |
| 采集模式 | 倾斜采集 |
| 倾斜 GSD | **1 厘米或以下** |
| 云台角度 | **45°** |
| 旁向重叠率 | **85%** |
| 航向重叠率 | **85%** |
| 飞行高度 | 高于测区内建筑最高高度 |

### 操作流程

1. 启动无人机，在遥控器内设置坐标系统为 WGS84。
2. 在遥控器地图上**框选测区范围**，选择倾斜采集模式。
3. 设置上述参数后，点击开始，飞机自动按航线循航飞行。
4. 飞行时**时刻关注飞机动态**，如有风险即时暂停并调整航线。

---

## 六、起降拍摄技巧

### 拍摄要点

1. 航线飞行完成后，前往每个起降点进行**起降拍摄**。
2. 每次起降时，将镜头对准**远处**、空中倾斜摄影和灵光扫描仪**都能看到的一个固定物体**。
3. 起降高度范围：
   - **最低点**：灵光扫描仪地面采集高度
   - **最高点**：航线设置平行的高度
4. 从低到高拍摄约 **30–40 张照片**，每次起降拍摄一组，相邻照片**重叠度不低于 85%**。
5. 每个起降点建议选择**三个不同方向**进行起降拍摄，以保证空地融合的成功率。

---

## 七、LCC Studio 数据处理

1. 打开 LCC Studio，点击「**马上生成**」。
2. 选择「**空地融合**」。
3. 输入项目名称。
4. 导入**地面数据**和**无人机照片**。
5. 点击「**开始**」进行空地融合模型数据处理。

---

## 八、关键参数速查

| 项目 | 要求 |
|---|---|
| 坐标系统 | WGS84（地面 + 无人机统一） |
| 照片重叠度 | ≥ 85% |
| 照片分辨率 | ≥ 1.0 cm/pixel |
| GSD 设置 | ≤ 1 cm |
| 云台角度 | 45° |
| 旁向 / 航向重叠率 | 均 85% |
| 起降点数量 | 4–5 个 |
| 起降拍摄方向 | 每点 3 个方向 |
| 每组起降照片数 | 30–40 张 |
| 地面回环 | 每个起降点半径 2m 内回环 2 次 |

起降点位置的选择：路口转弯人行道处
如何升降拍摄三角度30-40张照片，并保持重复率85%以上
