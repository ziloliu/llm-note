# ADC（模数转换器）基本原理与概念 笔记

---

## 一、ADC 基本概念

### 1.1 术语定义

| 缩写 | 全称 | 含义 |
|------|------|------|
| **ADC** | Analog to Digital Converter | 模数转换器（硬件）/ 模数转换（功能） |
| **DAC** | Digital to Analog Converter | 数模转换器（硬件）/ 数模转换（功能） |
| A | Analog | 模拟 |
| D | Digital | 数字 |

### 1.2 模拟信号 vs 数字信号

| 特性 | 模拟信号 | 数字信号 |
|------|---------|---------|
| 波形 | **连续** | **离散** |
| 取值 | 无限精确（小数点后无限） | 有限个离散值 |
| 表示方式 | 电压、电流连续变化 | 0 和 1（高低电平） |
| 处理电路 | 模拟电路 | 数字电路（CPU） |
| 典型来源 | 传感器输出 | 单片机内部数据 |
| 典型值 | 2.65V（连续变化） | 0/1（3.3V/0V） |

### 1.3 ADC 的作用

```
外部物理世界                    单片机系统
┌───────────┐                ┌──────────────┐
│ 传感器采集  │    ADC         │ 数字处理      │
│ 模拟物理量  │ ──────────→   │ 0 和 1 信号    │
│（连续变化） │    转换        │（离散数字量）   │
└───────────┘                └──────────────┘

例：温度传感器 → 电压值 2.65V → 数字量（12位二进制码）
```

### 1.4 ADC 与 DAC

```
ADC（模数转换）：模拟量 → 数字量
  例：2.65V → 0xCCC（12位）

DAC（数模转换）：数字量 → 模拟量
  例：0xCCC → 2.65V

两者是互逆过程
```

## 二、ADC 工作原理（两个阶段）

### 2.1 总体流程

```
原始模拟信号（连续）
    │
    ↓
┌─────────────┐
│  ① 采样阶段   │  时间维度：每隔一段时间取一个瞬时值
│  (Sampling)  │  连续信号 → 离散信号
└──────┬──────┘
       ↓
┌─────────────┐
│  ② 量化阶段   │  幅度维度：将瞬时值转换为二进制代码
│(Quantization)│  模拟值 → 数字值
└──────┬──────┘
       ↓
数字信号（离散的二进制代码）
```

### 2.2 采样阶段（时间维度）

```
问题：模拟信号在时间上是连续的，无法逐一处理
解决：每隔 Δt 时间取一个瞬时值

原始信号（连续）：
  ~~~∧~~∨~~∧~~∨~~∧~~∨~~
    
采样后的信号（离散）：
    •       •       •
   / \     / \     / \
  /   \   /   \   /   \
 •     • •     • •     •

特点：连续信号 → 离散的时间序列
```

### 2.3 采样能否完美恢复原始信号

```
问题：采样得到的离散点能否完美恢复原始连续波形？

答案：在一定条件下可以 → 香农采样定理

原始信号可分解为不同频率正弦波的叠加（傅里叶变换）：

  f(t) = A₀ + A₁sin(ω₁t+φ₁) + A₂sin(ω₂t+φ₂) + ...

只要采样频率足够高，能捕获最高频率分量
→ 所有频率分量都能被采样到
→ 叠加后可完美恢复原始波形
```

### 2.4 香农采样定理

```
采样频率 fs 必须满足：

  fs ≥ 2 × f_max

其中：
  fs    = 采样频率（每秒采样次数）
  f_max = 信号中最高频率分量

只有满足此条件，才能从采样信号完美恢复原始信号
```

```
采样频率足够高（fs ≥ 2f_max）：
  原始：~~∧~~∨~~∧~~∨~~
  采样：  •     •     •     •
  恢复：~~∧~~∨~~∧~~∨~~  ← 完美恢复 ✅

采样频率不足（fs < 2f_max）：
  原始：~~∧~~∨~~∧~~∨~~
  采样：  •           •
  恢复：  ——————————  ← 丢失高频细节 ❌（混叠失真）
```

### 2.5 量化阶段（幅度维度）

```
问题：采样得到的瞬时值（如 2.65V）如何用 0/1 表示？

解决：将连续的电压值转换为二进制代码

  采样值 2.65V
      ↓
  量化器（ADC 电路）
      ↓
  二进制代码（如 12 位：1100 1100 1100 = 0xCCC）

本质：连续的电压值 → 离散的数字代码
```

### 2.6 量化精度问题

```
二进制能够表示的数值是有限的：
  → 整数：精度无缺失，只有范围限制
  → 小数：精度会有缺失（有限位数无法精确表示）

量化精度取决于：
  → 二进制位数越多 → 精度越高
  → 二进制位数越少 → 精度越低
```

## 三、ADC 主要参数

### 3.1 参数总览

| 参数 | 含义 | STM32 典型值 |
|------|------|:------------:|
| **参考电压** | 确定模拟信号幅度范围 | VREF+ = 3.3V, VREF- = 0V |
| **通道数** | 能同时处理的模拟输入路数 | 最多 16 个外部 + 2 个内部 |
| **采样率** | 每秒采样次数 | 最高 1 MSPS |
| **分辨率** | 数字信号位数 | **12 位** |
| **转换时间** | 完成一次转换的时间 | 采样时间 + 量化时间 |

### 3.2 参考电压（VREF）

```
作用：确定 ADC 能够测量的电压范围

  VREF+ = 3.3V（上限）
  VREF- = 0V  （下限）

  测量范围：0V ~ 3.3V

参考电压为 ADC 提供了"标尺"：
  → 数字量 0 对应 0V
  → 数字量 4095（2^12-1）对应 3.3V
```

### 3.3 通道数

```
通道 = 模拟信号输入通道

每个通道连接一个外部模拟信号源
多通道可以分别采集不同的模拟信号

通道数 = ADC 能同时采集的模拟信号路数

STM32F103：
  每个 ADC 最多 16 个外部通道 + 2 个内部通道
  内部通道：温度传感器、内部参考电压
```

### 3.4 采样率

```
采样率 = 每秒钟 ADC 能进行多少次采样

单位：SPS（Samples Per Second）或 Hz

  1 MSPS = 1,000,000 次/秒

采样率越高：
  → 能捕获更高频率的信号
  → 信号还原精度更高
  → 但对硬件要求也更高

与香农采样定理的关系：
  采样率 fs ≥ 2 × f_max
```

### 3.5 分辨率

```
分辨率 = ADC 输出数字量的位数

位数 n → 能表示 2^n 个离散级别

| 位数 | 离散级别 | 最小变化量（3.3V 参考） |
|:----:|:-------:|:----------------------:|
| 8 位 | 256 | 3.3V/256 ≈ 12.9mV |
| 10 位 | 1024 | 3.3V/1024 ≈ 3.2mV |
| **12 位** | **4096** | **3.3V/4096 ≈ 0.8mV** |
| 16 位 | 65536 | 3.3V/65536 ≈ 0.05mV |

STM32F103 的 ADC 为 12 位分辨率：
  → 2^12 = 4096 个离散级别
  → 最小分辨率 = 3.3V / 4096 ≈ 0.806mV
  → 数字量 0 = 0V
  → 数字量 4095 = 3.3V
```

### 3.6 转换时间

```
转换时间 = 采样时间 + 量化时间

① 采样时间（Sampling Time）
   → 对模拟信号进行采样所需时间
   → 可配置（越长越精确，但速度越慢）

② 量化时间（Conversion Time）
   → 将采样值转换为二进制代码所需时间
   → 由 ADC 时钟频率和分辨率决定

影响因素：
  → ADC 时钟频率越高 → 转换越快
  → 分辨率越高（位数越多）→ 转换越慢
```

### 3.7 分辨率与转换时间的关系

```
分辨率高（如 16 位）：
  → 精度高（能区分更小的电压变化）
  → 转换时间长（需要更多的时钟周期）

分辨率低（如 8 位）：
  → 精度低（区分能力弱）
  → 转换时间短（需要更少的时钟周期）

实际应用中需要在精度和速度之间做权衡
```

## 四、ADC 转换结果计算

### 4.1 数字量转模拟量

```
已知数字量 D（0 ~ 4095），求模拟电压 V：

  V = D × VREF / 2^n

  其中：
    VREF = 参考电压（3.3V）
    n = 分辨率位数（12）
    2^n = 4096

例：
  D = 2048 → V = 2048 × 3.3 / 4096 = 1.65V
  D = 4095 → V = 4095 × 3.3 / 4096 ≈ 3.3V
  D = 0    → V = 0V
```

### 4.2 模拟量转数字量

```
已知模拟电压 V，求数字量 D：

  D = V × 2^n / VREF

  D = V × 4096 / 3.3

例：
  V = 2.65V → D = 2.65 × 4096 / 3.3 ≈ 3289
  V = 1.0V  → D = 1.0 × 4096 / 3.3 ≈ 1241
  V = 0V    → D = 0
```

## 五、关键公式汇总

```
分辨率：    离散级别数 = 2^n
最小变化量：ΔV = VREF / 2^n

数字量→电压：V = D × VREF / 2^n
电压→数字量：D = V × 2^n / VREF

采样定理：  fs ≥ 2 × f_max

STM32F103（12 位 ADC，VREF = 3.3V）：
  离散级别 = 4096
  最小分辨率 ≈ 0.806mV
  D = V × 4096 / 3.3
  V = D × 3.3 / 4096
```

## 六、傅里叶变换简介

```
任何周期信号都可以分解为不同频率正弦波的叠加：

  f(t) = A₀ + A₁sin(ω₁t + φ₁) + A₂sin(ω₂t + φ₂) + ...

时间域函数 f(t)  ←→  频域函数 F(ω)
        傅里叶变换

意义：
  → 将复杂波形分解为简单正弦波
  → 只要采样频率够高，能捕获最高频率分量
  → 就能完美恢复原始波形
```

## 七、概念速查

| 概念     | 说明                            |
| ------ | ----------------------------- |
| ADC    | 模拟数字转换器，将模拟量转为数字量             |
| DAC    | 数字模拟转换器，将数字量转为模拟量             |
| 采样     | 时间维度的离散化，隔 Δt 取一个瞬时值          |
| 量化     | 幅度维度的离散化，将模拟值转为二进制代码          |
| 香农采样定理 | fs ≥ 2 × f_max，才能完美恢复原始信号     |
| 傅里叶变换  | 时域信号 → 频域信号（分解为正弦波叠加）         |
| 参考电压   | 确定 ADC 测量范围的基准电压              |
| 离散级别   | ADC 能区分的不同电压值的个数（2^n）         |
| 分辨率    | ADC 输出数字量的位数（12 位 = 4096 级）   |
| 采样率    | 每秒采样次数（SPS）                   |
| 转换时间   | 采样时间 + 量化时间                   |
| VREF   | Reference Voltage，参考电压        |
| MSPS   | Mega Samples Per Second，百万次/秒 |
| SPS    | Samples Per Second，每秒采样次数     |
| 混叠失真   | 采样频率不足导致的信号失真                 |
