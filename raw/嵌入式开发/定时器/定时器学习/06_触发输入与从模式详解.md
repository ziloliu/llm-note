# 定时器触发输入与从模式详解 笔记

---

## 一、触发控制器与触发信号

### 1.1 触发控制器位置

```
时钟源 → 触发控制器 → 预分频器 → 计数器 → ARR（溢出）
             ↑
         TRGI（触发输入）   TRGO（触发输出）
```

### 1.2 两大触发信号

| 信号 | 全称 | 方向 | 功能 |
|------|------|------|------|
| **TRGI** | Trigger In | **输入** | 外部信号控制本定时器行为 |
| **TRGO** | Trigger Out | **输出** | 本定时器信号输出控制其他外设 |

---

## 二、主模式与从模式

### 2.1 概念

| 模式 | 条件 | 含义 |
|------|------|------|
| **主模式** | 本定时器 TRGO → 其他定时器 TRGI | 我驱动别人，我是主动的 |
| **从模式** | 其他定时器 TRGO → 本定时器 TRGI | 别人驱动我，我是被动的 |

### 2.2 级联关系示意

```
定时器 1（主）                    定时器 2（从）
┌──────────┐                    ┌──────────┐
│  TIM1    │   TRGO ──→ ITR0    │  TIM2    │
│  主模式   │ ─────────────────→ │  从模式   │
│          │   (内部固定连线)     │          │
└──────────┘                    └──────────┘
我输出信号                        我接收信号
我驱动别人                        别人驱动我
```

> 内部连线是芯片出厂时固定连接的，不可更改。

## 三、TRGI 触发输入源（8 种，4 大类）

### 3.1 SMCR.TS 位配置

| TS[2:0] | 来源 | 类别 |
|:-------:|------|------|
| 000 | ITR0 | 内部触发（其他定时器） |
| 001 | ITR1 | 内部触发（其他定时器） |
| 010 | ITR2 | 内部触发（其他定时器） |
| 011 | ITR3 | 内部触发（其他定时器） |
| **100** | **TI1F_ED** | **通道 1 双边沿检测** |
| **101** | **TI1FP1** | **通道 1 滤波+极性后信号** |
| **110** | **TI2FP2** | **通道 2 滤波+极性后信号** |
| 111 | ETRF | 外部触发引脚信号 |

### 3.2 四大类详解

#### 第一类：内部触发（ITR0~3）

```
来源：其他定时器的 TRGO 输出
连接方式：芯片内部固定连线，不可更改
用途：定时器级联

信号路径：
  其他定时器 TRGO → 内部连线 → ITRx → 多路选择 → TRGI
```

#### 第二类：通道 1 双边沿（TI1F_ED）

```
来源：自身通道 1 引脚输入
特点：上升沿和下降沿都会产生信号（双边沿检测）
信号名称：TI1F_ED（Edge Detect）

信号路径：
  CH1 引脚 → 滤波器 → 双边沿检测 → TI1F_ED → TRGI

与其他的区别：
  TI1FP1/TI2FP2 只能选一种极性（单边沿）
  TI1F_ED 两个边沿都检测
```

#### 第三类：通道直通信号（TI1FP1 / TI2FP2）

```
来源：自身通道 1 或通道 2 引脚输入
特点：经过滤波和选定极性的边沿检测（单边沿）

信号路径：
  CH1 引脚 → 滤波器 → 边沿检测（极性选定）→ TI1FP1 → TRGI
  CH2 引脚 → 滤波器 → 边沿检测（极性选定）→ TI2FP2 → TRGI

⚠️ 本实验使用的触发源就是这一类！
```

#### 第四类：外部触发（ETRF）

```
来源：ETR 专用外部触发引脚
特点：独立于输入通道，有独立的滤波和预分频

信号路径：
  ETR 引脚 → 极性选择 → 预分频 → 滤波 → ETRF → TRGI
```

### 3.3 四类信号对比

| 类别 | TS 值 | 来源 | 信号名 | 特点 |
|------|:-----:|------|--------|------|
| 内部触发 | 000~011 | 其他定时器 | ITR0~3 | 内部固定连线 |
| 双边沿 | 100 | CH1 引脚 | TI1F_ED | 上升+下降沿 |
| 直通信号 | 101/110 | CH1/CH2 引脚 | TI1FP1/TI2FP2 | 单边沿（极性选定）|
| 外部触发 | 111 | ETR 引脚 | ETRF | 独立输入通路 |

## 四、ITR 内部连接表

### 4.1 连接关系（从定时器视角）

| 从定时器 | ITR0 (TS=000) | ITR1 (TS=001) | ITR2 (TS=010) | ITR3 (TS=011) |
|:--------:|:-------------:|:-------------:|:-------------:|:-------------:|
| TIM2 | TIM1 | TIM8 | TIM3 | TIM4 |
| TIM3 | TIM1 | TIM2 | TIM5 | TIM4 |
| TIM4 | TIM1 | TIM2 | TIM3 | TIM8 |
| TIM5 | TIM2 | TIM3 | TIM4 | TIM8 |

### 4.2 查表说明

```
例：TIM2 选择 ITR0 → TRGI 来自 TIM1
    → TIM1 是主，TIM2 是从

例：TIM4 选择 ITR3 → TRGI 来自 TIM8
    → TIM8 是主，TIM4 是从

注意：不能自己触发自己
  TIM2 的 ITR0~3 不会连到 TIM2
  TIM3 的 ITR0~3 不会连到 TIM3
```

### 4.3 级联示意图

```
TIM1 TRGO ──→ TIM2 ITR0
         ──→ TIM3 ITR0
         ──→ TIM4 ITR0

TIM2 TRGO ──→ TIM3 ITR1
         ──→ TIM5 ITR0

TIM3 TRGO ──→ TIM4 ITR2
         ──→ TIM5 ITR1

TIM8 TRGO ──→ TIM2 ITR1
         ──→ TIM4 ITR3
         ──→ TIM5 ITR3
```

## 五、从模式（SMCR.SMS，8 种模式）

### 5.1 SMCR 寄存器

```
SMCR（从模式控制寄存器）：
  SMS[2:0] = 从模式选择（3 位，8 种模式）
  TS[2:0]  = 触发源选择（3 位，8 种来源）
```

### 5.2 八种从模式

| SMS[2:0] | 模式 | 功能 | 计数器行为 |
|:--------:|------|------|-----------|
| **000** | **关闭** | 不使用从模式 | 内部时钟正常计数 |
| 001 | 编码器模式 1 | TI1FP1 计数，TI2FP2 边沿 | 仅 TI1 计数 |
| 010 | 编码器模式 2 | TI2FP2 计数，TI1FP1 边沿 | 仅 TI2 计数 |
| 011 | 编码器模式 3 | TI1FP1 + TI2FP2 都计数 | 两个都计数 |
| **100** | **复位模式** | TRGI 上升沿 → **CNT 清零** | 复位 |
| **101** | **门控模式** | TRGI 高=使能，低=停止 | 开关控制 |
| **110** | **触发模式** | TRGI 上升沿 → 启动计数 | 仅启动 |
| **111** | **外部时钟** | TRGI 上升沿 → CNT 加 1 | 替换时钟源 |

### 5.3 各模式详解

#### SMS=000：关闭从模式（默认）

```
→ TRGI 信号被忽略
→ 计数器由内部时钟驱动
→ 就是我们之前一直使用的模式
```

#### SMS=001~011：编码器模式

```
→ TI1FP1 和 TI2FP2 直接连接编码器模块
→ 根据两个信号的相位关系自动加/减计数
→ 用于旋转编码器位置检测
→ 本实验不使用
```

#### SMS=100：复位模式（本实验使用）

```
→ TRGI 信号的上升沿 → CNT 自动清零
→ 同时产生更新寄存器的信号

关键：
  ✅ 硬件自动清零，无需中断
  ✅ 无需软件干预
  ✅ 清零时刻精确（硬件响应）
```

#### SMS=101：门控模式

```
→ TRGI 高电平 → 计数器使能（开始计数）
→ TRGI 低电平 → 计数器停止（不复位，暂停）

特点：
  → 相当于外部信号控制计数器开关
  → 停止后不会清零，再次开启继续计数
  → 类似于使能位 CEN，但由外部信号控制
```

#### SMS=110：触发模式

```
→ TRGI 上升沿 → 启动计数器
→ 不做复位，只是启动
→ 用于延迟启动计数器

特点：
  → 只负责启动，不负责停止
  → 启动后持续计数
```

#### SMS=111：外部时钟模式

```
→ TRGI 上升沿 → CNT 加 1
→ TRGI 直接替代内部时钟驱动计数器
→ 这是唯一改变时钟源的模式

⚠️ 与其他模式的区别：
  其他模式：内部时钟驱动计数，TRGI 控制行为（复位/使能等）
  此模式：TRGI 本身就是计数时钟
```

### 5.4 模式对比总结

| 对比项 | 关闭 | 复位 | 门控 | 触发 | 外部时钟 |
|--------|:----:|:----:|:----:|:----:|:--------:|
| 计数时钟 | 内部 | 内部 | 内部 | 内部 | **TRGI** |
| TRGI 作用 | 无 | 清零 CNT | 开/关 | 启动 | 计数 |
| CNT 复位 | ❌ | ✅ 上升沿 | ❌ | ❌ | ❌ |
| CNT 使能控制 | ❌ | ❌ | ✅ 高/低 | ✅ 仅启动 | ❌ |

## 六、为什么本实验选择复位模式 + TI1FP1

### 6.1 选择触发源：TI1FP1（TS=101）

```
原因：
  → 需要检测 PWM 信号的上升沿来清零计数器
  → TI1FP1 是通道 1 经过滤波和极性选择后的信号
  → 极性设为上升沿 → 每个上升沿产生 TRGI 信号

信号路径：
  CH1 引脚(PB6) → 滤波器 → 上升沿检测 → TI1FP1 → TRGI
```

### 6.2 选择从模式：复位模式（SMS=100）

```
原因：
  → 测量周期：两个上升沿之间的时间
  → 每个上升沿自动清零 CNT → CNT 从零开始计数
  → 下一个上升沿到来前 CCR1 自动捕获 CNT → 周期值
  → 每个下降沿自动捕获 CNT 到 CCR2 → 高电平时间

优势：
  ✅ 无需中断
  ✅ 无需全局变量
  ✅ 硬件自动完成，精度更高
```

### 6.3 与之前中断方式的对比

| 对比项 | 中断方式 | 复位模式 |
|--------|---------|---------|
| CNT 清零 | 软件（中断服务函数） | **硬件（TRGI 上升沿）** |
| 响应速度 | 延迟若干时钟周期 | **瞬间（1 个周期）** |
| 需要中断 | ✅ | ❌ |
| 需要全局变量 | ✅（最初版本） | ❌ |
| 精度 | 受中断延迟影响 | **更高** |

## 七、寄存器配置要点

### 7.1 SMCR 寄存器配置

| 位域 | 位 | 值 | 功能 |
|------|:--:|:--:|------|
| TS[2:0] | 6:4 | **101** | 触发源选择 TI1FP1 |
| SMS[2:0] | 2:0 | **100** | 从模式选择复位模式 |

```c
// 配置示例
TIM4->SMCR |= (0b101 << 4);   // TS = 101（TI1FP1）
TIM4->SMCR |= (0b100 << 0);   // SMS = 100（复位模式）
```

### 7.2 完整信号路径

```
CH1 引脚 (PB6)
    │
    ↓
滤波器 (IC1F=0，不滤波)
    │
    ↓
边沿检测 (CC1P=0，上升沿)
    │
    ↓
TI1FP1
    │
    ├──→ CCR1 捕获（直连，CC1S=01）→ 测周期
    │
    └──→ TRGI（TS=101）
              │
              ↓
         从模式控制器（SMS=100，复位模式）
              │
              ↓
         CNT 清零（硬件自动）
```

## 八、概念速查

| 概念 | 说明 |
|------|------|
| **TRGI** | 触发输入信号，控制本定时器行为 |
| **TRGO** | 触发输出信号，输出到其他定时器/外设 |
| **主模式** | 本定时器 TRGO 驱动别人 |
| **从模式** | 别人 TRGO 驱动本定时器（通过 TRGI） |
| **SMCR.TS** | 触发源选择（8 种 TRGI 来源） |
| **SMCR.SMS** | 从模式选择（8 种工作模式） |
| **复位模式** | TRGI 上升沿 → CNT 硬件自动清零 |
| **门控模式** | TRGI 高电平使能，低电平停止 |
| **触发模式** | TRGI 上升沿启动计数 |
| **外部时钟** | TRGI 替代内部时钟驱动计数 |
| **ITR** | 内部触发连线，连接其他定时器 TRGO |
| **TI1FP1** | 通道 1 滤波+极性后的信号 |
| **TI2FP2** | 通道 2 滤波+极性后的信号 |
| **TI1F_ED** | 通道 1 双边沿检测信号 |
| **ETRF** | 外部触发引脚信号 |
