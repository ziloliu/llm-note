# ADC 工作原理 — 逐次逼近型详解 笔记

---

## 一、ADC 常见类型对比

### 1.1 四种主要类型

| 类型 | 核心原理 | 速度 | 精度 | 电路复杂度 | 典型应用 |
|------|---------|:----:|:----:|:---------:|---------|
| **逐次逼近型（SAR）** | 逐位二分法比较 | **中等** | **中等** | **中等** | STM32、通用MCU |
| **Sigma-Delta（Σ-Δ）** | 差值累加积分 | 慢 | **高** | 较高 | 高精度测量、音频 |
| **双积分型** | 两次积分 | 慢 | **高** | 较低 | 高抗干扰测量 |
| **Flash（闪速）型** | 多比较器并行比较 | **极快** | 低 | 高 | 高速采样、视频 |

### 1.2 各类型原理简介

**逐次逼近型（SAR）**
```
原理：二分法，从最高位到最低位逐位尝试
  → 取中间值与待测电压比较
  → 大了就往下找，小了就往上找
  → 每次缩小一半范围
  → n 位需要 n 次比较

特点：速度与精度的均衡选择
STM32 选用此类型
```

**Sigma-Delta（Σ-Δ）型**
```
原理：对相邻采样值的差值（Δ）进行累加（Σ）
  → Σ（Sigma）= 累加
  → Δ（Delta）= 差值/增量
  → 底层有积分器

特点：高精度，但转换速度慢
  → 排除外界干扰能力强
  → 适合高精度测量和音频处理

衍生：双积分型
  → 两次积分，抗干扰更强
  → 精度更高，速度更慢
```

**Flash（闪速）型**
```
原理：电阻分压 + 多个比较器并行比较
  → 参考电压通过电阻串分出多个电位
  → 每个电位与输入电压同时比较
  → 所有位同时得出结果

特点：速度极快（1个时钟周期完成）
  → 但精度低（位数少）
  → 电路规模大（每个电位需一个比较器+编码器）
  → 精度提高 → 电路规模指数增长
```

### 1.3 STM32 选择逐次逼近型的原因

```
精度够用：12 位分辨率 → 4096 级
速度适中：不像 Σ-Δ 型那么慢
集成度高：电路规模合理，适合片上集成

→ 精度与速度之间的最佳平衡
```

## 二、天平称重类比（核心理解）

### 2.1 天平称重过程

```
砝码组：5g, 4g, 3g, 2g, 1g
测量范围：1g ~ 15g（全部加在一起）
待测物体：6g（未知重量）

左盘 = 物体（待测电压 Ux）
右盘 = 砝码（DAC 输出电压）
```

### 2.2 逐步操作

```
口诀：先上大，后上小

步骤 1：放 5g    → 左盘重（物体 > 5g）→ 保留 5g
步骤 2：放 4g    → 右盘重（5+4=9 > 6g）→ 去掉 4g
步骤 3：放 3g    → 右盘重（5+3=8 > 6g）→ 去掉 3g
步骤 4：放 2g    → 右盘重（5+2=7 > 6g）→ 去掉 2g
步骤 5：放 1g    → 平衡！（5+1=6 = 6g）→ 保留 1g

结果：5g + 1g = 6g ✅
```

### 2.3 关键规则

```
① 从最大的砝码开始
② 放上后比较：
   → 偏小（物体更重）→ 保留当前砝码，继续加下一个
   → 偏大（砝码更重）→ 去掉刚加的砝码，换更小的
③ 只操作最后新上的那个砝码
④ 已经确定的砝码不再修改
```

## 三、逐次逼近型 ADC 硬件结构

### 3.1 核心组件框图

```
                     逻辑控制电路
                   (控制逐位保留/清除)
                     ↑         ↓
                    比较结果    控制信号
                     ↑         ↓
  Ux(33V) → (+) ┌─────────┐  8位寄存器
                │  比 比较器 │←──┐    │
  DAC输出 → (-) └─────────┘   │    ↓
                     ↓         │  ┌──────────┐
                   输出(0/1)   └──│ 8位DAC    │←── VREF(255V)
                                  │ (数→模转换)│
                                  └──────────┘
                                       ↓
                                  输出缓冲区 → 结果
```

### 3.2 组件功能说明

| 组件 | 功能 | 天平类比 |
|------|------|---------|
| **比较器** | 比较 Ux 与 DAC 输出，输出高/低电平 | 天平（左盘=物体，右盘=砝码） |
| **8 位寄存器** | 存储当前二进制砝码值 | 当前放在天平上的砝码组 |
| **DAC 转换器** | 将寄存器数字量转为模拟电压 | 计算砝码总重量 |
| **逻辑控制电路** | 根据比较结果决定保留/清除当前位 | 操作砝码增减的人 |
| **VREF** | 参考电压，确定量程 | 砝码最大量程 |
| **输出缓冲区** | 存储最终转换结果 | 称量结果 |

## 四、比较器工作原理

```
两个输入端：
  (+) 正端：待测电压 Ux
  (-) 负端：DAC 输出电压（砝码电压）

输出规则：
  Ux > DAC 输出 → 正端更大 → 输出高电平（1）
    → 砝码偏小 → 保留当前位 = 1

  Ux < DAC 输出 → 负端更大 → 输出低电平（0）
    → 砝码偏大 → 清除当前位 = 0

  Ux = DAC 输出 → 平衡 → 当前值即为结果

本质：模拟电压 → 高低电平（0/1）→ 控制数字逻辑
```

## 五、8 位寄存器与砝码对应关系

### 5.1 简化模型参数

```
为了方便计算：
  VREF = 255V（实际不可能，纯粹为了好算）
  分辨率 = 8 位
  数字量范围 = 0 ~ 255
  全零 → 0V
  全一（255）→ 255V

  数字值 = 模拟电压值（一对一对应）
  待测电压 Ux = 33V
```

### 5.2 位权砝码表

| 位 | 寄存器置 1 时的值 | 十进制（砝码重量） | 模拟电压 |
|:--:|:----------------:|:----------------:|:--------:|
| Bit 7（最高位） | 1000 0000 | **128** | 128V |
| Bit 6 | 0100 0000 | **64** | 64V |
| Bit 5 | 0010 0000 | **32** | 32V |
| Bit 4 | 0001 0000 | **16** | 16V |
| Bit 3 | 0000 1000 | **8** | 8V |
| Bit 2 | 0000 0100 | **4** | 4V |
| Bit 1 | 0000 0010 | **2** | 2V |
| Bit 0（最低位） | 0000 0001 | **1** | 1V |

### 5.3 与天平砝码的区别

```
天平砝码：5, 4, 3, 2, 1（等差递减 1）
ADC 砝码：128, 64, 32, 16, 8, 4, 2, 1（等比递减一半）

ADC 砝码 = 2 的幂次递减 → 每次缩小一半 → 二分法
```

## 六、DAC 转换器（内部组件）

```
ADC 内部集成了一个 DAC（数模转换器）：

作用：将寄存器中的数字量转为模拟电压
输入：8 位数字量 + VREF
输出：对应的模拟电压

例：寄存器 = 0b10000001 = 129
    VREF = 255V
    DAC 输出 = 129V（因为 VREF 与全一值一一对应）
```

---

## 七、逻辑控制电路

```
逻辑控制电路 = 操作砝码增减的人

判断逻辑：
  比较器输出 = 高电平（Ux > DAC）→ 砝码偏小
    → 保留当前位（1）
    → 移到下一位，置 1 继续尝试

  比较器输出 = 低电平（Ux < DAC）→ 砝码偏大
    → 清除当前位（0）
    → 移到下一位，置 1 继续尝试

关键原则：
  → 从最高位开始，逐位向低位推进
  → 每次只操作当前正在尝试的那一位
  → 上一位已确定，不再修改
  → 每次新尝试：将下一位 置 1
```

## 九、逐次逼近型 ADC 工作流程总结

```
① 初始化：寄存器清零，准备最高位（Bit n-1）

② 将当前位 置 1

③ DAC 将寄存器值转换为模拟电压

④ 比较器比较 Ux 与 DAC 输出：
   → 高电平（Ux > DAC）：砝码偏小 → 保留当前位 = 1
   → 低电平（Ux < DAC）：砝码偏大 → 清除当前位 = 0

⑤ 移到下一位（n-2），重复 ②③④

⑥ 直到最低位（Bit 0）判断完毕

⑦ 寄存器中的值即为最终转换结果

⑧ 结果放入输出缓冲区

时间复杂度：n 位分辨率 → n 次比较
  8 位 → 8 次比较
  12 位（STM32）→ 12 次比较
```

## 十、从 8 位推广到 12 位（STM32）

```
STM32 参数：
  分辨率：12 位
  VREF+ = 3.3V
  VREF- = 0V
  离散级别：2^12 = 4096
  最小分辨率：3.3V / 4096 ≈ 0.806mV

转换过程：
  → 从 Bit11（最高位）开始
  → 逐位判断到 Bit0（最低位）
  → 共需 12 次比较
  → 每次范围缩小一半

例：输入电压 2.65V
  D = 2.65 × 4096 / 3.3 ≈ 3289
  二进制：0b110011011001
```

## 十一、关键公式

```
① 离散级别数 = 2^n
   n = 8  → 256
   n = 12 → 4096

② 最小分辨率（量化步长）ΔV = VREF / 2^n

③ 数字量 → 电压：V = D × VREF / 2^n

④ 电压 → 数字量：D = V × 2^n / VREF

⑤ 转换比较次数 = n（n 位需要 n 次比较）

⑥ 逐次逼近本质 = 二分搜索
   每次比较将搜索范围缩小一半
```

## 十二、三种类型核心差异总结

```
逐次逼近型（SAR）：
  1 个比较器 + 1 个 DAC + 逻辑控制
  逐位比较，n 次完成
  速度与精度均衡 ⭐（STM32 选用）

Sigma-Delta 型：
  1 个比较器 + 积分器 + 数字滤波器
  累积差值，高精度低速
  Σ=累加，Δ=差值

Flash 型：
  2^n - 1 个比较器并行
  1 个时钟周期完成
  速度快但精度低、电路庞大
```

## 十三、概念速查

| 概念 | 说明 |
|------|------|
| 逐次逼近 | 从高位到低位逐位比较确定值 |
| 二分法 | 每次比较将搜索范围缩小一半 |
| 比较器 | 比较两个输入电压大小，输出高低电平 |
| DAC | 数字模拟转换器，将寄存器值转为模拟电压 |
| VREF | 参考电压，确定 ADC 量程范围 |
| 量化步长 | 最小分辨率，ΔV = VREF / 2^n |
| 位权 | 每一位对应的权重（2 的幂次：128,64,32,16,8,4,2,1） |
| 逻辑控制电路 | 控制逐位判断，决定保留或清除当前位 |
| 输出缓冲区 | 存储最终转换结果 |
| 天平类比 | 左盘=待测电压，右盘=DAC输出，砝码=位权 |
