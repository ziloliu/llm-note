# PWM（脉冲宽度调制）原理详解 笔记

---

## 一、什么是 PWM

| 项目 | 说明 |
|------|------|
| 全称 | Pulse Width Modulation（脉冲宽度调制） |
| 别名 | 脉宽调制 |
| 本质 | 利用**数字信号**控制**模拟电路**的技术 |
| 核心 | 通过调节方波的**高电平占比（占空比）**改变输出功率 |

> PWM 不是数模转换（DAC），DAC 是真正输出模拟电压值；PWM 输出的是数字方波，只是通过面积等效来模拟模拟量的效果。

---

## 二、LED 呼吸灯需求

```
目标：LED 亮度从暗 → 最亮 → 暗，循环往复，如同呼吸

问题：
  数字输出只能输出 0（0V）或 1（3.3V）
  无法直接输出中间电压值（如 1.65V、0.25V）

解决方案：
  PWM → 快速切换高低电平 → 人眼视觉暂留 → 看起来是持续的中间亮度
```

## 三、PWM 的核心原理

### 3.1 面积等效法则

```
┌────────── 3.3V ──────────┐
│ ████████████████████████  │     阴影面积 A
└──────────────────────────┘

┌── 3.3V ──┐              ┌── 3.3V ──┐
│ ████████ │              │ ████████ │    阴影面积 B
└──────────┘              └──────────┘

如果 面积 A = 面积 B，则两者的输出功率等效
```

```
原理：
  两个形状不同的信号，如果它们的"面积"相同
  → 对于惯性负载（电机、LED+人眼），表现效果相同
  → 等效于一个恒定的中间电压值

示例：
  3.3V 方波，占空比 50% → 等效输出 ≈ 1.65V
  3.3V 方波，占空比 25% → 等效输出 ≈ 0.825V
  3.3V 方波，占空比 75% → 等效输出 ≈ 2.475V
```

### 3.2 "惯性"的含义

```
PWM 能起作用的前提：被控制对象具有"惯性"

惯性 = 状态不会突变

不同对象的惯性来源：
  ┌─────────────┬──────────────────────────────┐
  │   对象       │     惯性来源                  │
  ├─────────────┼──────────────────────────────┤
  │   电机       │   物理惯性（转子质量）         │
  │   灯泡       │   热惯性（灯丝温度）          │
  │   LED 灯     │   人眼视觉暂留（~100ms）       │
  └─────────────┴──────────────────────────────┘
```

### 3.3 人眼视觉暂留

```
人眼视觉暂留时间 ≈ 0.1 秒（100ms）

  PWM 频率 > 10Hz → 人眼开始感受不到闪烁
  PWM 频率 > 24Hz → 基本感受不到闪烁（动画原理）
  PWM 频率 > 50Hz → 完全无闪烁感

  → 只要 PWM 频率足够高，快速开关 LED
  → 人眼看到的就是持续的中间亮度
  → 占空比决定了"中间亮度"的高低
```

## 四、PWM 三大关键参数

### 4.1 参数总览

| 参数 | 符号 | 说明 | 影响 |
|------|:----:|------|------|
| **周期** | T | 一个完整波形的重复时间 | 决定频率 |
| **频率** | f | f = 1/T，每秒重复次数 | 人眼是否能看出闪烁 |
| **占空比** | D | 高电平时间 t / 周期 T | **决定等效输出功率/亮度** |

### 4.2 周期与频率

```
周期 T：连续两个上升沿（或下降沿）之间的时间

    ┌───┐         ┌───┐
    │   │         │   │
────┘   └─────────┘   └─────
    |←─ T ─→|
    |←─ T ─→|

频率 f = 1 / T

示例：
  T = 10ms  → f = 100Hz
  T = 1ms   → f = 1000Hz = 1kHz
  T = 20ms  → f = 50Hz
```

### 4.3 频率对亮度的影响

```
⚠️ 频率不会影响亮度！

频率只决定高低电平切换的速度
  → 频率太低 → 人眼看出闪烁 → 无法模拟连续亮度
  → 频率足够高 → 人眼感觉不到闪烁 → 可以模拟亮度
  → 频率再高 → 亮度不变（占空比不变的情况下）

频率定好之后一般不再改变
```

### 4.4 占空比（核心参数）

```
占空比 D = 高电平时间 t / 周期 T × 100%

    ┌───┐         ┌───┐
    │   │         │   │
────┘   └─────────┘   └─────
    |t|             
    |←── T ──→|

占空比 = t / T

示例：
  D = 50% → 高电平占一半 → 等效输出 ≈ 50% 最大亮度
  D = 25% → 高电平占 1/4 → 等效输出 ≈ 25% 最大亮度
  D = 75% → 高电平占 3/4 → 等效输出 ≈ 75% 最大亮度
  D = 0%  → 全低电平      → LED 熄灭
  D = 100% → 全高电平     → LED 最亮
```

### 4.5 占空比与亮度关系

```
占空比 0%  ──── ████████░░░░░░░░░░░░  LED 熄灭
占空比 25% ──── ████████▓▓░░░░░░░░░░  暗
占空比 50% ──── ████████▓▓▓▓▓▓░░░░░░  中等亮度
占空比 75% ──── ████████▓▓▓▓▓▓▓▓▓▓░░  较亮
占空比 100% ─── ████████▓▓▓▓▓▓▓▓▓▓▓▓  最亮

（█ = 高电平持续时间，░ = 低电平持续时间）
```

## 五、PWM 的典型应用场景

| 应用 | 控制对象 | 调节的物理量 | 频率典型值 |
|------|----------|-------------|-----------|
| **呼吸灯** | LED | 亮度 | 1kHz~10kHz |
| **电机调速** | 直流电机 | 转速 | 1kHz~20kHz |
| **舵机控制** | 舵机 | 角度 | 50Hz |

| **蜂鸣器** | 无源蜂鸣器 | 音调 | 2kHz~5kHz |

## 七、PWM 与 DAC 的区别

| 对比项 | PWM | DAC |
|--------|-----|-----|
| 输出信号 | **数字方波** | **模拟电压** |
| 输出值 | 只有 0 和 3.3V | 连续可调（如 0~3.3V） |
| 原理 | 面积等效 | 真正的数模转换 |
| 精度 | 较低（受频率和分辨率限制） | 较高 |
| 成本 | **低**（只需一个 GPIO） | 较高（需要 DAC 硬件） |
| 适用场景 | 电机、LED、蜂鸣器 | 音频、精密模拟输出 |

## 八、PWM 生成原理（通用定时器）

### 8.1 输出比较生成 PWM

```
ARR 决定 PWM 周期（频率）
CCR 决定高电平持续时间（占空比）

计数器 CNT 持续递增：
  → CNT < CCR 时，输出高电平
  → CNT ≥ CCR 时，输出低电平

  占空比 = CCR / (ARR + 1) × 100%
```

### 8.2 PWM 波形生成过程

```
ARR = 99（周期 = 100 个计数）
CCR = 30（占空比 = 30%）

CNT: 0  1  2  ...  29  30  31  ...  99  0  1  ...
                  ↑           ↑              ↑
输出: ██████████████████████████████░░░░░░░░░░░░░░░░░░░░████
      |← CCR=30 高电平 →|← ARR+1-CCR=70 低电平 →|

占空比 = 30 / 100 = 30%
```

### 8.3 调节占空比

```
固定 ARR（频率不变），改变 CCR：

CCR = 0   → 占空比 = 0%   → LED 熄灭
CCR = 25  → 占空比 = 25%  → LED 较暗
CCR = 50  → 占空比 = 50%  → LED 半亮
CCR = 75  → 占空比 = 75%  → LED 较亮
CCR = 100 → 占空比 = 100% → LED 最亮
```

## 九、呼吸灯实现思路

```
① 配置定时器输出 PWM（固定频率，如 1kHz）
② 循环改变 CCR 值（占空比）
   → CCR 从 0 逐渐增大到 ARR → LED 渐亮
   → CCR 从 ARR 逐渐减小到 0 → LED 渐暗
③ 循环往复 → 呼吸效果

伪代码：
while(1)
{
    for (CCR = 0; CCR <= ARR; CCR++)   // 渐亮
    {
        TIM->CCR = CCR;
        Delay_ms(小段时间);
    }
    for (CCR = ARR; CCR >= 0; CCR--)   // 渐暗
    {
        TIM->CCR = CRR;
        Delay_ms(小段时间);
    }
}
```

## 十、总结

| 概念 | 说明 |
|------|------|
| **PWM** | 用数字方波模拟模拟量的技术 |
| **周期 T** | 一个完整波形的重复时间 |
| **频率 f** | 1/T，决定人眼是否看出闪烁 |
| **占空比 D** | 高电平时间/周期，**决定等效输出功率** |
| **面积等效** | 面积相同的信号等效输出功率相同 |
| **惯性** | PWM 起作用的前提（物理惯性或人眼视觉暂留） |
| **呼吸灯** | 通过周期性改变占空比实现 LED 亮度渐变 |
